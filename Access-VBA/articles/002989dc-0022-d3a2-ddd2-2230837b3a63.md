
# RDS Properties

 **Last modified:** March 09, 2015

 _ **Applies to:** Access 2013 | Access 2016_

|||
|:-----|:-----|
|[Connect (RDS)](11aa3284-18e9-6d2d-761b-c25090370b77.md)|Indicates the database name from which the query and update operations are run.|
|[ExecuteOptions (RDS)](fb244cbd-9a03-9128-1373-694c9061c9da.md)|Indicates whether asynchronous execution is enabled.|
|[FetchOptions (RDS)](0d86c5e4-9abc-5c0e-dc04-4183f4c278cc.md)|Indicates the type of asynchronous fetching.|
|[FilterColumn (RDS)](fb5d9f23-b62a-8131-d6ff-8b7ed8bb825c.md)|Indicates the column on which to evaluate the filter criteria.|
|[FilterCriterion (RDS)](51e6cb64-a404-114e-8e1a-0744cceeec3e.md)|Indicates the evaluation operator to use in the filter value.|
|[FilterValue (RDS)](66dc14cd-cc14-78cb-cb05-91eefb17ea47.md)|Indicates the value to filter records.|
|[Handler (RDS)](aaf8c8c6-f95b-3cf3-b3f6-203f37464c87.md)|Indicates the name of a server-side customization program ( _handler_ ) that extends the functionality of the **RDSServer.DataFactory**, and any parameters used by the _handler_.|
|[InternetTimeout (RDS)](66fc6e87-3d23-ce2c-18f5-0fc83ac43801.md)|Indicates the number of milliseconds to wait before a request times out.|
|[ReadyState (RDS)](e7b62205-a604-ef43-2f5d-9b51b46d2b5a.md)|Indicates the progress of a  **DataControl** object as it fetches data into its **Recordset** object.|
|[Recordset and SourceRecordset (RDS)](5f4bb72d-ddfa-41c0-c353-b3a6632b4a91.md)|Indicates the  **Recordset** object returned from a custom business object.|
|[Server (RDS)](17519dbe-a43a-1d0d-22c1-dc0def2f63ab.md)|Indicates the Internet Information Services (IIS) name and communication protocol.|
|[SortColumn (RDS)](0a5d157c-9261-960d-6f89-33d9c94b3940.md)|Indicates by which column to sort the records.|
|[SortDirection (RDS)](33de0dce-f371-6a54-d179-0627939f5b14.md)|Indicates whether a sort order is ascending or descending.|
|[SQL (RDS)](http://msdn.microsoft.com/library/210adcbb-5c89-150b-4c61-6a52dea9af56%28Office.15%29.aspx)|Indicates the query string used to retrieve the  **Recordset**.|
|[URL (RDS)](722765dc-f89c-0131-73b1-69c56a795546.md)|Indicates a string that contains a relative or absolute URL.|
