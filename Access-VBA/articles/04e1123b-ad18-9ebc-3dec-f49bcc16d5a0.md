
# CurrentProject.IsConnected Property (Access)

You can use the  **IsConnected** property to determine if the **[CurrentProject](e6baae73-1eeb-b48f-d35e-b3e921378561.md)** object is currently connected. Read-only **Boolean**.


## Syntax

 _expression_. **IsConnected**

 _expression_ A variable that represents a **CurrentProject** object.
