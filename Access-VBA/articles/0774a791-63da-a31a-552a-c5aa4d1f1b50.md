
# Workspaces.Delete Method (DAO)

 **Last modified:** March 09, 2015

 _ **Applies to:** Access 2013 | Access 2016_

Deletes the specified  **Workspace** form the **Workspaces** collection.


## Syntax

 _expression_. **Delete**( ** _Name_** )

 _expression_ A variable that represents a **Workspaces** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Name_|Required|**String**|The name of the workspace to delete.|
