
# Line Control

The line control displays a horizontal, vertical, or diagonal line on a form or report.


## Remarks

You can use  **Border Width** to change the line width. You can use **Border Color** to change the color of the border or make it transparent. You can change the line style (dots, dashes, and so on) of the border by using the **[BorderStyle](1DDF622C-B1C4-009B-06DF-0A6BC819B367.md)** property.
