
# DoCmd.RefreshRecord Method (Access)

The  **RefreshRecord** method performs the **RefreshRecord** macro operation from Visual Basic.


## Syntax

 _expression_. **RefreshRecord**

 _expression_ A variable that represents a **DoCmd** object.


## Remarks

In a client database, the  **RefreshRecord** method is equivalent to the **[Refresh](e7a15c34-d3ec-184f-8d03-3e264fcc60d0.md)** method of the **Form** object. In a web database, see the **[RefreshRecord](http://msdn.microsoft.com/library/68c90d7d-f59c-9e83-bc30-8f37cf5a3696%28Office.15%29.aspx)** macro action.
