
# ListBox.VerticalAnchor Property (Access)

Gets or sets an [AcVerticalAnchor](08f16c8b-1566-cfad-795a-cb65a91c4e52.md) constant that indicates how the specified list box is anchored vertically within its layout. Read/write.


## Syntax

 _expression_. **VerticalAnchor**

 _expression_ A variable that represents a **ListBox** object.


## Remarks

The default value is  **acVerticalAnchorTop**.
