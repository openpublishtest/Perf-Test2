
# ObjectFrame.UpdateMethod Property (Access)

This property has been deprecated. Use the  **[UpdateOptions](29effba2-7427-62ca-c0d6-6ed5081b0e02.md)** property to specify how a linkedOLE object is updated.


## Syntax

 _expression_. **UpdateMethod**

 _expression_ A variable that represents an **ObjectFrame** object.
