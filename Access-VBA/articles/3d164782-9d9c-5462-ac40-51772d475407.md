
# TextBox.TextFormat Property (Access)

Gets or sets whether rich text is displayed in the specified text box. Read/write [AcTextFormat](cce0f7f5-ec7d-b80b-71a4-95052b6b7af1.md).


## Syntax

 _expression_. **TextFormat**

 _expression_ A variable that represents a **TextBox** object.
