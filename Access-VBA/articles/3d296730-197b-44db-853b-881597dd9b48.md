
# / (Divide) (Access custom web app)
Divides one number by another.

 **Last modified:** June 29, 2012

 _ **Applies to:** Access 2013 | Access 2016_

 **In this article**
[Syntax](#sectionSection0)
[Return Type](#sectionSection1)
[Remarks](#sectionSection2)



## Syntax
<a name="sectionSection0"> </a>

 _dividend_ / _divisor_

 _dividend_ Is the numeric expression to divide. Can be any valid expression of any one of the data types of the numeric data type category, except the datetime data type.

 _Divisor_ Is the numeric expression by which to divide the dividend. Can be any valid expression of any one of the data types of the numeric data type category, except the datetime data type.


## Return Type
<a name="sectionSection1"> </a>

Returns the data type of the argument with the higher precedence. 

If an integer  _dividend_ is divided by an integer _divisor_, the result is an integer that has any fractional part of the result truncated.


## Remarks
<a name="sectionSection2"> </a>

The actual value returned by the / operator is the quotient of the first expression divided by the second expression.
