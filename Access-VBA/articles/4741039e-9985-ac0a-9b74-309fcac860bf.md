
# TextBox.DisplayAsHyperlink Property (Access)

Gets or sets an  **[AcDisplayAsHyperlink](fb9d9af3-9aff-3031-2f94-6715211d6ee4.md)** constant that specifies whether to display the contents of the specified text box as a hyperlink. Read/write.


## Syntax

 _expression_. **DisplayAsHyperlink**

 _expression_ A variable that represents a **TextBox** object.
