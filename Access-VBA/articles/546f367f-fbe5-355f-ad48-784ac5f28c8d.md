
# Form.FilterOnLoad Property (Access)

Gets or sets whether the filter specified by the  **[Filter](5eb49f82-8519-981c-a663-9862736ac95f.md)** property is applied when the form is loaded. Read/write **Boolean**.


## Syntax

 _expression_. **FilterOnLoad**

 _expression_ A variable that represents a **Form** object.
