
# ComboBox.SeparatorCharacters Property (Access)

Gets or sets the separator displayed between values when the combo box is bound to a multi-valued field. Read/write [AcSeparatorCharacters](f6f0c881-d092-b68a-ea1f-bd4e28b730a8.md).


## Syntax

 _expression_. **SeparatorCharacters**

 _expression_ A variable that represents a **ComboBox** object.
