
# Form.OrderByOnLoad Property (Access)

Gets or sets whether the sorting specified by the  **[OrderBy](6ca9c25e-9f16-1f08-1ac3-6f19761f9f55.md)** property is applied when the form is loaded. Read/write **Boolean**.


## Syntax

 _expression_. **OrderByOnLoad**

 _expression_ A variable that represents a **Form** object.
