
# Attachment.DisplayAs Property (Access)

Gets or sets an  **[AcDisplayAs](ccd8275b-d3b5-e3b1-3b0c-cad08ff71887.md)** constant that specifies how a file should be displayed in the **Attachment** control. Read/write.


## Syntax

 _expression_. **DisplayAs**

 _expression_ A variable that represents an **Attachment** object.
