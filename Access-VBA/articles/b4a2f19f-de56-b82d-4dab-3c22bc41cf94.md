
# ComboBox.Layout Property (Access)

Returns the type of layout for the specified combo box. Read-only  **[AcLayoutType](ee963ed0-9293-8ad8-5694-4b93a5e4d89a.md)**.


## Syntax

 _expression_. **Layout**

 _expression_ A variable that represents a **ComboBox** object.
