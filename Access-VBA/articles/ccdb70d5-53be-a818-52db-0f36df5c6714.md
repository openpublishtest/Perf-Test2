
# Label.HorizontalAnchor Property (Access)

Gets or sets an  **[AcHorizontalAnchor](2b9f0574-252d-7957-d25d-cb382d2cee73.md)** constant that indicates how the label is anchored horizontally within its layout. Read/write.


## Syntax

 _expression_. **HorizontalAnchor**

 _expression_ A variable that represents a **Label** object.


## Remarks

The default value is  **acHorizontalAnchorLeft**.
