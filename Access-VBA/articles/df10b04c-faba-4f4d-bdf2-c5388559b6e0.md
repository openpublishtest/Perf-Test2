
# Report Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Circle](4f5d24e2-75bf-3586-7e0d-0902adee61a6.md)|The  **Circle** method draws a circle, an ellipse, or an arc on a **Report** object when the Print event occurs.|
|[Line](9e640e37-c055-3dc3-b70e-0805cdc13561.md)|The  **Line** method draws lines and rectangles on a **Report** object when the Print event occurs.|
|[Move](169ac85a-394f-5db2-7b55-b6ca5fd03546.md)|Moves the specified object to the coordinates specified by the argument values.|
|[Print](6f8523cc-7b17-ec27-e2c9-a7ae3d5a8c3f.md)|The  **Print** method prints text on a **[Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)** object using the current color and font.|
|[PSet](951a262b-b17b-9b95-b5f2-922d4aff9ce9.md)|The  **PSet** method sets a point on a **[Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)** object to a specified color when the **Print** event occurs.|
|[Requery](d078d523-3bbd-fa70-44ac-116cdcedfebd.md)|The  **Requery** method updates the data underlying the specified report by requerying the source of data for the control.|
|[Scale](6a261d1d-9474-7374-f399-4d46e404058b.md)|The  **Scale** method defines the coordinate system for a **[Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)** object.|
|[TextHeight](cac67d4c-e140-06ae-ccbd-961cdee3d087.md)|The  **TextHeight** method returns the height of a text string as it would be printed in the current font of a **[Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)** object.|
|[TextWidth](98827373-8610-5e48-ab46-2c89f8e2d2a7.md)|The  **TextWidth** method returns the width of a text string as it would be printed in the current font of a **[Report](6f77c1b4-a9ce-7caa-204c-fe0755c6f9df.md)** object.|
