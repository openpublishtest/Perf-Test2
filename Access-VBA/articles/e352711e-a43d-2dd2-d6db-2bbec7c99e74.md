
# Form.RibbonName Property (Access)

Gets or set the name of the customized ribbon to be displayed when the specified form is loaded. Read/write  **String**.


## Syntax

 _expression_. **RibbonName**

 _expression_ A variable that represents a **Form** object.


## Remarks

For more information about how to display a customized ribbon when loading a form, see [How to: Apply a Filter When Opening a Form or Report](d7a43e62-3003-d411-2128-dffe0536e119.md).
