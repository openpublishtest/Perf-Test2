
# Text file specification field separator matches decimal separator or text delimiter. (Error 3441)

Two or more of the following delimiters settings used by the Text ISAM are the same:



- TxtDelimiter
    
- NumberDecimalSymbol
    
- CurrencyDecimalSymbol
    
- Format
