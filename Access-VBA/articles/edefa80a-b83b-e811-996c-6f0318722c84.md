
# UpdateRule Property (ADOX)

 **Last modified:** June 29, 2011

 _ **Applies to:** Access 2013 | Access 2016_



Indicates the action performed when a primary key is updated.

## Settings and Return Values

Sets and returns a  **Long** value that can be one of the[RuleEnum](5b59f202-315b-09b7-8505-9ac08ceccb3d.md) constants. The default value is **adRINone**.


## Remarks

This property is read-only on [Key](727198ec-57d2-7766-790c-370beb931de6.md) objects already appended to the collection.
