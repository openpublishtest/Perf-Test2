
# SlicerCaches.Count Property (Excel)

Returns the number of objects in the collection.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that returns a **[SlicerCaches](d6097f70-cdc7-3be7-575c-cf43a0765e10.md)** collection.


### Return Value

 **Long**
