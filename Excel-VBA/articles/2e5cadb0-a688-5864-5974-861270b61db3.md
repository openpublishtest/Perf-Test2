
# Worksheet.Sort Property (Excel)

Returns a  **[Sort](637ee681-743c-5196-2bfc-4a5bea025295.md)** object. Read-only.


## Syntax

 _expression_ . **Sort**

 _expression_ A variable that represents a **Worksheet** object.
