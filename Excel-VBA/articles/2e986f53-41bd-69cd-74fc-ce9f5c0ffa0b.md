
# Slicers.Item Property (Excel)

Returns a  **[Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object from the collection. Read-only.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that represents a **[Slicers](12b67ff5-cf66-35d1-2c72-9aa2f4a396a0.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|The name or index number of the  **Slicer** object.|
