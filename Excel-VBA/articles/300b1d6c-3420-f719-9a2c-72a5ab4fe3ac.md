
# Model.Name Property (Excel)

Returns a  **String** value representing the name of the **Model** object. Read-only


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a[Model Members (Excel)](7946bddc-7c4a-3519-52c8-526af2b55ef3.md) object.


## Property value

 **STRING**
