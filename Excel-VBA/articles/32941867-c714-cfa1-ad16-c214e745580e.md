
# DisplayFormat.Locked Property (Excel)

Returns a value that indicates if the associated  **[Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object is locked as it is displayed in the current user interface. Read-only.


## Syntax

 _expression_ . **Locked**

 _expression_ A variable that represents a **[DisplayFormat](c70b5d7f-adf1-e539-a32d-12c920af7c7e.md)** object.


### Return Value

Variant


## Remarks

Returns  **True** if the range is locked, **False** if the range can be modified when the sheet is protected, or **Null** if the range contains both locked and unlocked cells.
