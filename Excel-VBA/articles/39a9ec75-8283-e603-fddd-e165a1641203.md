
# ChartGroup.BinsUnderflowValue Property (Excel)

If an [BinsUnderflowEnabled](719d315a-c3ed-77e9-3b42-0f6300b6bf8d.md) is **True**, specifies the value below which an underflow bin is displayed. Read/write  **Double**.


## Syntax

 _expression_ . **BinsUnderflowValue**

 _expression_ A variable that represents a **ChartGroup** object.
