
# ChartGroup.BinsOverflowEnabled Property (Excel)

Specifies whether a bin for values above the [BinsOverflowValue](411856a7-ac17-e9eb-35bd-c851c0cfdfdc.md) is enabled. Read/write **Boolean**.


## Syntax

 _expression_ . **BinsOverflowEnabled**

 _expression_ A variable that represents a **ChartGroup** object.
