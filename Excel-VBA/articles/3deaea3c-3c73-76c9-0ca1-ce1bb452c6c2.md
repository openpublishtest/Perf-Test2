
# ProtectedViewWindow.Close Method (Excel)

Closes the specified  **Protected View** window.


## Syntax

 _expression_ . **Close**

 _expression_ A variable that represents a **[ProtectedViewWindow](6a32240c-c90b-c51a-6f8e-c3ff496b9855.md)** object.


### Return Value

Boolean
