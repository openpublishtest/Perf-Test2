
# ModelFormatCurrency.Symbol Property (Excel)

Specifies the symbol to use to represent the currency. Read/write  **String**.


## Syntax

 _expression_ . **Symbol**

 _expression_ A variable that represents a **ModelFormatCurrency** object.
