
# ChartGroup.BinsUnderflowEnabled Property (Excel)

Specifies whether a bin for values below the [BinsUnderflowValue](39a9ec75-8283-e603-fddd-e165a1641203.md) is enabled. Read/write **Boolean**.


## Syntax

 _expression_ . **BinsUnderflowEnabled**

 _expression_ A variable that represents a **ChartGroup** object.
