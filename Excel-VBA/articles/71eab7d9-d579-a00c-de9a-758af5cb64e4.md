
# DropLines.Border Property (Excel)

Returns a  **[Border](bca516bf-7c0f-f9df-078d-dfb522f256f3.md)** object that represents the border of the object.


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents a **DropLines** object.
