
# ChartGroup.BinsType Property (Excel)

Specifies how the horizontal axis of the histogram chart is formatted, by bins type. Read/write [XlBinsType](99482ffa-a40c-c2b4-a062-ce5ce2ad5b29.md).


## Syntax

 _expression_ . **BinsType**

 _expression_ A variable that represents a **ChartGroup** object.
