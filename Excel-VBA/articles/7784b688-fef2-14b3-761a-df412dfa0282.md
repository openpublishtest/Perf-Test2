
# OLEObject.Update Method (Excel)

Updates the link.


## Syntax

 _expression_ . **Update**

 _expression_ A variable that represents an **OLEObject** object.


### Return Value

Variant


## Example

This example updates the link to OLE object one on Sheet1.


```
Worksheets("Sheet1").OLEObjects(1).Update
```
