## Example

This example fills the union of two named ranges, Range1 and Range2, with the formula =RAND().


```vb
Worksheets("Sheet1").Activate 
Set bigRange = Application.Union(Range("Range1"), Range("Range2")) 
bigRange.Formula = "=RAND()"
```
