
# SlicerItem.Parent Property (Excel)

Returns the parent  **[SlicerCache](6e6533e3-0503-a1d3-9ecd-f7997233565f.md)** object for the slicer item. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

SlicerCache
