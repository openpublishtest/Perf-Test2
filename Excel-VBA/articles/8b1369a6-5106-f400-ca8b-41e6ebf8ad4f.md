
# SparklineGroup.Delete Method (Excel)

Deletes the sparkline group.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

Nothing
