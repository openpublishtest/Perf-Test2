
# Action.Type Property (Excel)

Returns the action type. Read-only  **[XlActionType](9a4489d3-6342-f86d-5eec-f9db2852f7c0.md)** .


## Syntax

 _expression_ . **Type**

 _expression_ A variable that represents an **Action** object.
