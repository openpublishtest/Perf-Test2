
# ValueChange.Tuple Property (Excel)

Returns the MDX tuple of the value was changed in the OLAP data source. Read-only


## Syntax

 _expression_ . **Tuple**

 _expression_ A variable that represents a **[ValueChange](27335d52-7003-2268-b5d0-c2cd21588579.md)** object.


### Return Value

 **String**
