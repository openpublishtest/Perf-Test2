
# SlicerCaches.Item Property (Excel)

Returns a single  **[SlicerCache](6e6533e3-0503-a1d3-9ecd-f7997233565f.md)** object from the collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that returns a **[SlicerCaches](d6097f70-cdc7-3be7-575c-cf43a0765e10.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|The name or index number of the object.|

### Return Value

 **SlicerCache**
