
# SparklineGroup.Axes Property (Excel)

Returns the associated  **[SparkAxes](bcd36a3c-772e-3317-b22e-27447ce23e5b.md)** object. Read-only.


## Syntax

 _expression_ . **Axes**

 _expression_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

SparkAxes
