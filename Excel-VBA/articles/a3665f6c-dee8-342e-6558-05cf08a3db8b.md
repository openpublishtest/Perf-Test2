
# Slicer.Delete Method (Excel)

Deletes the slicer and removes it from the associated  **[Slicers](12b67ff5-cf66-35d1-2c72-9aa2f4a396a0.md)** collection.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **[Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object.


### Return Value

Nothing
