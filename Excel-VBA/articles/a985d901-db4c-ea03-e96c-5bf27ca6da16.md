
# SparklineGroups.Clear Method (Excel)

Clears the selected sparklines.


## Syntax

 _expression_ . **Clear**

 _expression_ A variable that represents a **[SparklineGroups](9bc6be34-fa2e-8652-ca92-fa9630b4d7a6.md)** object.


### Return Value

Nothing
