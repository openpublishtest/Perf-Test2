
# SparklineGroup.Count Property (Excel)

Returns the number of sparklines in the sparkline group. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

Long
