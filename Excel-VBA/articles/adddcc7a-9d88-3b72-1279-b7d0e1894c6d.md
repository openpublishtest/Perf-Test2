
# Point.PictureUnit2 Property (Excel)

Returns or sets the unit for each picture on the chart if the  **[PictureType](6adfa1b4-a37d-1a20-fbab-806d007d1ebf.md)** property is set to **xlStackScale** (if not, this property is ignored). Read/write **Double** .


## Syntax

 _expression_ . **PictureUnit2**

 _expression_ A variable that represents a **Point** object.
