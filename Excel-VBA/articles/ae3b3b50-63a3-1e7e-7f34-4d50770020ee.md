
# RTD.RestartServers Method (Excel)

Reconnects to a real-time data server (RTD).


## Syntax

 _expression_ . **RestartServers**

 _expression_ A variable that represents a **[RTD](50f289c3-081b-108b-2fee-c4069a04a8e7.md)** object.
