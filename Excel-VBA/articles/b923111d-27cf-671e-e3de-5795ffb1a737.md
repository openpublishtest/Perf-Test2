
# SlicerCache.Index Property (Excel)

 Returns the index of the specified **[SlicerCache](6e6533e3-0503-a1d3-9ecd-f7997233565f.md)** object in the **[SlicerCaches](d6097f70-cdc7-3be7-575c-cf43a0765e10.md)** collection. Read-only


## Syntax

 _expression_ . **Index**

 _expression_ A variable that represents a **SlicerCache** object.
