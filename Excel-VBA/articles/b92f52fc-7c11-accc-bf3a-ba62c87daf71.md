
# Model.ModelMeasures Property (Excel)

Returns a [ModelMeasures Object (Excel)](b0edac9a-e10d-ec51-d9e7-6fa8a29dcda8.md) object that represents the collection of model measures in the data model. Read-only.


## Syntax

 _expression_ . **ModelMeasures**

 _expression_ A variable that represents a **Model** object.
