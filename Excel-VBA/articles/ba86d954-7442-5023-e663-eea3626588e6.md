
# Series.ParentDataLabelOption Property (Excel)

Specifies the parent data label option (banner, overlapping, or none) for the specified series within the chart group. Read/write [XLParentDataLabelOptions](eb2c2212-e538-e6a4-2a76-c14808ff679c.md).


## Syntax

 _expression_ . **ParentDataLabelOption**

 _expression_ A variable that represents a **Series** object.
