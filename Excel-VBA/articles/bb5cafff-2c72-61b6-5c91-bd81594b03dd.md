
# Slicers.Count Property (Excel)

Returns the number of objects in the collection.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Slicers](12b67ff5-cf66-35d1-2c72-9aa2f4a396a0.md)** collection.


### Return Value

 **Long**
