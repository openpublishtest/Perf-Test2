
# ModelRelationship.Active Property (Excel)

When  **True** , the relationship is active. When **False** , this relationship is inactive. **Boolean** . Read/Write


## Syntax

 _expression_ . **Active**

 _expression_ A variable that represents a[ModelRelationship Object (Excel)](8b0a7fad-06a5-178d-c5b2-96fc5528a3cc.md) object.


## Property value

 **BOOL**
