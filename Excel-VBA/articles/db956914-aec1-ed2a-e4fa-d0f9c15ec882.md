
# Scenarios.Merge Method (Excel)

Merges the scenarios from another sheet into the  **[Scenarios](90d6ff4b-f329-a04c-040e-a39bb501a58b.md)** collection.


## Syntax

 _expression_ . **Merge**( **_Source_** )

 _expression_ A variable that represents a **Scenarios** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Source_|Required| **Variant**|The name of the sheet that contains scenarios to be merged, or a  **[Worksheet](182b705e-854a-81cc-a4b0-59b942de55ae.md)** object that represents that sheet.|

### Return Value

Variant


## Remarks

The value of a merged range is specified in the cell of the range's upper-left corner.
