
# SortFields.Clear Method (Excel)

Clears all the  **[SortFields](a9c83ea1-1cd9-1552-1f03-71bd92a2cc72.md)** objects.


## Syntax

 _expression_ . **Clear**

 _expression_ A variable that represents a **SortFields** object.
