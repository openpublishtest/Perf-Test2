
# Model.ModelFormatBoolean Property (Excel)

Returns a [ModelFormatBoolean](b6a43c30-1dd9-39e0-86dc-fd229bb51c87.md) object that represents formatting of type true/false in the data model. Read-only .


## Syntax

 _expression_ . **ModelFormatBoolean**

 _expression_ A variable that represents a **Model** object.
