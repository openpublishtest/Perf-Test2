
# ChartFormat.AutoShapeType Property (Excel)

Returns the type of the specified shape.  **MsoAutoShapeType** Read-only


## Syntax

 _expression_ . **AutoShapeType**

 _expression_ A variable that represents a **ChartFormat** object.


## Property value

 **MSOAUTOSHAPETYPE**
