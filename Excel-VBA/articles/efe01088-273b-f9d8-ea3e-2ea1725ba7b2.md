
# Workbook.ModelChange Event (Excel)

Occurs after the Excel data model is changed. 


## Syntax

 _expression_ . **ModelChange**_(Changes)_

 _expression_ A variable that represents a **Workbook** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Changes_|Required|MODELCHANGES|[ModelChanges Object (Excel)](fd2388eb-48ab-c238-2ffa-8c3f6d20fe36.md) object which includes changes made to the Excel data model during the last transaction.|
