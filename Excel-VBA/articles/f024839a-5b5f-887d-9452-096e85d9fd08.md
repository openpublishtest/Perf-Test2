
# SlicerItems.Item Property (Excel)

Returns a  **[SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object from a collection. Read-only.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that represents a **[SlicerItems](80bbbbab-711a-cefb-255b-94fe2994d3c8.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|The name or index number of the  **SlicerItem** object.|

### Return Value

SlicerItem
