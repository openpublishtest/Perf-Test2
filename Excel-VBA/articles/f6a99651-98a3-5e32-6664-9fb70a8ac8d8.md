
# SlicerItems.Parent Property (Excel)

Returns the parent  **[SlicerCache](6e6533e3-0503-a1d3-9ecd-f7997233565f.md)** object for the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[SlicerItems](80bbbbab-711a-cefb-255b-94fe2994d3c8.md)** object.


### Return Value

SlicerCache
