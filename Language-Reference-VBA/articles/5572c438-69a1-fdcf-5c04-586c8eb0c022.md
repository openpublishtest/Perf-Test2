
# Toggle Breakpoint Command (Debug Menu)

Sets or removes a [breakpoint](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) at the current line. You can't set a breakpoint on lines containing nonexecutable code such as comments, declaration statements, or blank lines.

A line of code in which a breakpoint is set appears in the colors specified in the  **Editor Format** tab of the **Options** dialog box.

Toolbar shortcut: 
![](images/tbr_bkpt_ZA01201681.gif). Keyboard shortcut: F9.
