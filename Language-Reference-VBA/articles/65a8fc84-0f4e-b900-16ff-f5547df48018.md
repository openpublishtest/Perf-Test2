
# InsertFile Method



Inserts code from a file into a code module.
 _object_**.InsertFile(**_filename_**) As String**
The  **InsertFile** syntax has these parts:


|**Part**|**Description**|
|:-----|:-----|
| _object_|Required. An [object expression](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) that evaluates to an object in the Applies To list.|
| _filename_|Required. A [String](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) specifying the file containing the code to insert into the code module.|
