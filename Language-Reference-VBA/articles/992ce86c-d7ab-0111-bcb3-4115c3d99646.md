
# Calendar Constants

The following [constants](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) can be used anywhere in your code in place of the actual values:



|**Constant**|**Value**|**Description**|
|:-----|:-----|:-----|
|**vbCalGreg**|0|Indicates that the Gregorian calendar is used.|
|**vbCalHijri**|1|Indicates that the Hijri calendar is used.|
