
# ReplaceLine Method (VBA Add-In Object Model)



Replaces an existing line of code with a specified line of code.
 **Syntax**
 _object_**.ReplaceLine(**_line_, _code_**)**
The  **ReplaceLine** syntax has these parts:


|**Part**|**Description**|
|:-----|:-----|
| _object_|Required. An [object expression](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) that evaluates to an object in the Applies To list.|
| _line_|Required. A [Long](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) specifying the location of the line you want to replace.|
| _code_|Required. A [String](b8bdf64f-5920-1ae9-16d0-b26d09524a30.md) containing the code you want to insert.|
