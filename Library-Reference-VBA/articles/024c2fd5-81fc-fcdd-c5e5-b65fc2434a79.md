
# CustomXMLPart.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLPart** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLPart** object.


### Return Value

Object
