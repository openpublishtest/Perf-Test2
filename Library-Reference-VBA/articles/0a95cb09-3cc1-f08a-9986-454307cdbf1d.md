
# Font2.Shadow Property (Office)

Gets the value specifying the type of shadow effect for the selection of text. Read-only.


## Syntax

 _expression_. **Shadow**

 _expression_ An expression that returns a **Font2** object.


### Return Value

ShadowFormat
