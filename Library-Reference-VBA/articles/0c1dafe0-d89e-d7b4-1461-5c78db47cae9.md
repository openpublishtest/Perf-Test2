
# WorkflowTasks.Count Property (Office)

Gets a  **Long** indicating the number of items in the **WorkflowTasks** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **WorkflowTasks** object.


### Return Value

Long
