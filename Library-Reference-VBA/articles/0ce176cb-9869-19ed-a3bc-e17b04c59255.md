
# SignatureSet.Subset Property (Office)

Gets or sets a value that acts as a filter on the available  **Signature** objects for a document. Read/write.


## Syntax

 _expression_. **Subset**

 _expression_ An expression that returns a **SignatureSet** object.


### Return Value

MsoSignatureSubset
