
# BulletFormat2.Font Property (Office)

Gets a  **Font2** object that represents character formatting for a **BulletFormat2** object. Read-only.


## Syntax

 _expression_. **Font**

 _expression_ An expression that returns a **BulletFormat2** object.


### Return Value

Font2
