
# ODSOColumn.Parent Property (Office)

Gets the  **Parent** object for the **ODSOColumn** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **ODSOColumn** object.


### Return Value

Object
