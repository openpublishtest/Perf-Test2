
# DocumentProperties.Item Property (Office)

Gets a  **DocumentProperty** object from the **DocumentProperties** collection. Read-only.


## Syntax

 _expression_. **Item**( ** _Index_** )

 _expression_ Required. A variable that represents a **[DocumentProperties](90d42786-7d9a-b604-dbdf-88db41cbe69b.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Variant**|The name or index number of the document property returned.|
