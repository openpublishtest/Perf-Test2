
# BulletFormat2.Visible Property (Office)

Gets or sets a value that specifies whether the bullet is visible. Read/write.


## Syntax

 _expression_. **Visible**

 _expression_ An expression that returns a **BulletFormat2** object.


### Return Value

MsoTriState
