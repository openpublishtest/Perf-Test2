
# TextRange2.Count Property (Office)

Gets a  **Long** indicating the number of items in the **TextRange2** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **TextRange2** object.


### Return Value

Long
