
# CommandBarButton.Style Property (Office)

Gets or sets the way a  **CommandBarButton** control is displayed. Read/write.


## 


 **Note**  The use of CommandBars in some Microsoft Office applications has been superseded by the new ribbon component of the Microsoft Office Fluent user interface. For more information, search Help for the keyword "ribbon."


## Syntax

 _expression_. **Style**

 _expression_ A variable that represents a **[CommandBarButton](e6d8209d-2c87-f1b5-bc3f-d4e5e5d3ab73.md)** object.


## Remarks

The value of the  **Style** property is available as a value in the **[MsoButtonStyle](da6a0258-5cef-1745-793d-ead1ac4e9006.md)** enumeration.
