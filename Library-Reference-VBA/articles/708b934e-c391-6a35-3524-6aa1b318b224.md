
# FileDialogFilters.Delete Method (Office)

Removes a file dialog filter.


## Syntax

 _expression_. **Delete**( ** _filter_** )

 _expression_ Required. A variable that represents a **[FileDialogFilters](a74663cf-ad63-e41a-8d5e-e51e8a20c173.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _filter_|Optional|**Variant**|The filter to be removed.|
