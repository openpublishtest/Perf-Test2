
# SoftEdgeFormat.Radius Property (Office)

Gets or sets the size, measured in points, of the soft edge effect of the shape. Read/write


## Syntax

 _expression_. **Radius**

 _expression_ An expression that returns a **SoftEdgeFormat** object.
