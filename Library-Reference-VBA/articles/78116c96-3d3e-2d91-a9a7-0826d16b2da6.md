
# DocumentInspectors.Count Property (Office)

Gets the number of items in the  **DocumentInspectors** object. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **DocumentInspectors** object.


### Return Value

Integer
