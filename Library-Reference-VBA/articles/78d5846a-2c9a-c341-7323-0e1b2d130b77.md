
# TabStops2.DefaultSpacing Property (Office)

Gets or sets the default spacing between tab stops. Read/write.


## Syntax

 _expression_. **DefaultSpacing**

 _expression_ An expression that returns a **TabStops2** object.


### Return Value

Single
