
# SharedWorkspaceLink.Delete Method (Office)

Deletes the current  **SharedWorkspaceLink** object.


 **Note**  Beginning with Microsoft Office 2010, this object or member has been deprecated and should not be used.


## Syntax

 _expression_. **Delete**

 _expression_ Required. A variable that represents a **[SharedWorkspaceLink](eb36dbed-fc41-08df-3cbc-affbaf5f9784.md)** object.
