
# TextColumn2.Spacing Property (Office)

Gets or sets the spacing between text columns in a  **TextColumn2** object. Read/write.


## Syntax

 _expression_. **Spacing**

 _expression_ An expression that returns a **TextColumn2** object.


### Return Value

Single
