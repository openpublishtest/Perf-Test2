
# PictureEffect.Type Property (Office)

Specifies the type of Picture Effect. Read-only


## Syntax

 _expression_. **Type**

 _expression_ An expression that returns a **PictureEffect** object.


## Remarks

This property takes a  **MsoPictureEffectType** enumeration.
