
# GlowFormat.Color Property (Office)

Gets a  **ColorFormat** object that represents the color of text formatted as glow. Read-only.


## Syntax

 _expression_. **Color**

 _expression_ An expression that returns a **GlowFormat** object.


### Return Value

ColorFormat
