
# ThemeColor.RGB Property (Office)

Gets or sets a value of a color in the color scheme of a Microsoft Office theme. Read/write.


## Syntax

 _expression_. **RGB**

 _expression_ An expression that returns a **ThemeColor** object.


### Return Value

MsoRGBType
