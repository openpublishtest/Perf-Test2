
# ODSOColumns.Parent Property (Office)

Gets the  **Parent** object for the **ODSOColumns** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **ODSOColumns** object.


### Return Value

Object
