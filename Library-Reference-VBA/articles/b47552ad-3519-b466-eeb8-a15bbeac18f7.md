
# ParagraphFormat2.Bullet Property (Office)

Gets a  **BulletFormat2** object for the paragraph. Read-only.


## Syntax

 _expression_. **Bullet**

 _expression_ An expression that returns a **ParagraphFormat2** object.


### Return Value

BulletFormat2
