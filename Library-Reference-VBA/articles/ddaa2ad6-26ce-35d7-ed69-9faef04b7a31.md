
# Signature.IsSigned Property (Office)

Gets a  **Boolean** value indicating whether the document was signed successfully. Read-only.


## Syntax

 _expression_. **IsSigned**

 _expression_ An expression that returns a **Signature** object.


### Return Value

Boolean
