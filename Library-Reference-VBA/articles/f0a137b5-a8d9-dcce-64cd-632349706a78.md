
# TabStops2.Parent Property (Office)

Gets an  **Object** that represents the **Parent** object of the specified **TabStops2** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **TabStops2** object.


### Return Value

Object
