
# CustomXMLNodes.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLNodes** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLNodes** object.


### Return Value

Object
