
# ContactItem.Gender Property (Outlook)

Returns or sets an  **[OlGender](5d0db92c-276c-bf50-b192-c33d7aee5409.md)** constant indicating the gender of the contact. Read/write.


## Syntax

 _expression_ . **Gender**

 _expression_ A variable that represents a **ContactItem** object.
