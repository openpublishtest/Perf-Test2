
# OutlookBarStorage Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](ad9a37cb-de9c-662e-1dcd-2197ebd239d2.md)|Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent Outlook application for the object. Read-only.|

|[Parent](d8f82919-4a68-4092-f37b-87d63b0dd57a.md)|Returns the parent  **Object** of the specified object. Read-only.|
