
# OlkTimeZoneControl.Value Property (Outlook)

Returns or sets a  **Variant** that represents the content of the control. Read/write.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents an **OlkTimeZoneControl** object.
