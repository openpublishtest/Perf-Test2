
# Folder.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **Folder** object.


## Remarks

The parent of the  **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** object is the **[Folders](0c814c3c-74fc-414c-982d-a0097fcb35c2.md)** collection object.
