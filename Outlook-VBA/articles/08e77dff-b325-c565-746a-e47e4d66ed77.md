
# ContactItem.ReminderOverrideDefault Property (Outlook)

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.


## Syntax

 _expression_ . **ReminderOverrideDefault**

 _expression_ A variable that represents a **ContactItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the **[ReminderPlaySound](a9941154-6c65-57c7-1dab-6d6a59620d92.md)** and the **[ReminderSoundFile](aafbdc5b-816f-3605-d265-5da349e9e791.md)** properties.
