
# FormRegion.Close Event (Outlook)

Represents the closing of the frame for a form region, which occurs just before the  **[Close](5a83b3d3-6096-9e37-88b1-00f97c0bf8bd.md)** event of the Inspector associated with the Outlook item.


## Syntax

 _expression_ . **Close**

 _expression_ A variable that represents a **FormRegion** object.
