
# RuleConditions.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **RuleConditions** object.


## Remarks

The parent of the  **[RuleConditions](e8e9a05a-b36b-add2-b294-8cdc5a97e119.md)** object is the **[Rule](ea2ddbcc-fd65-a636-c6da-79950033f385.md)** object.
