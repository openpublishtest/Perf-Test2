
# CalendarModule.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **CalendarModule** object.


## Remarks

This property returns  **olCalendarModule** for the **[CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** object.
