
# ConversationHeader.Session Property (Outlook)

Returns the  **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** object for the current session. Read-only.


## Syntax

 _expression_ . **Session**

 _expression_ A variable that represents a **[ConversationHeader](5142d5f7-55c1-4d9d-3a11-d25c8763fcb7.md)** object.


## Remarks

Returns  **Null** ( **Nothing** in Visual Basic) if there is no logged-on session.

You can use the  **Session** property and the **[GetNamespace](6175d0d9-5a61-ce45-35c0-b70895d757b3.md)** method interchangeably to obtain the **NameSpace** object for the current session.
