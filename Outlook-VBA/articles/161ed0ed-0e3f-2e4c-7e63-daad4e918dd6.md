
# MailItem.TaskDueDate Property (Outlook)

Returns or sets a  **Date** value that represents the due date of the task for this **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** . Read/write.


## Syntax

 _expression_ . **TaskDueDate**

 _expression_ An expression that returns a **MailItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the **[IsMarkedAsTask](6cc4530d-fa74-916b-654d-db995d9a989f.md)** property is set to **False** .
