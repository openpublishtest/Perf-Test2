
# TableView.DefaultExpandCollapseSetting Property (Outlook)

Returns or sets an  **[OlDefaultExpandCollapseSetting](b05310dc-0bb6-2f78-d3f2-56c02402bbf9.md)** constant that determines the default expansion setting for groups in the **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object. Read/write.


## Syntax

 _expression_ . **DefaultExpandCollapseSetting**

 _expression_ A variable that represents a **TableView** object.
