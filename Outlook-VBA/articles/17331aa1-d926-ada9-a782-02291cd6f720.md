
# ExchangeUser.Application Property (Outlook)

Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent application (Outlook) for the **[ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)** object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents an **ExchangeUser** object.
