
# ExchangeUser.Parent Property (Outlook)

Returns the parent  **Object** of the **[ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)** object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **ExchangeUser** object.
