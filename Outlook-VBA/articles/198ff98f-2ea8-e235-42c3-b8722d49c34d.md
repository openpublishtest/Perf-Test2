
# DistListItem.ClearTaskFlag Method (Outlook)

Clears the  **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** object as a task.


## Syntax

 _expression_ . **ClearTaskFlag**

 _expression_ An expression that returns a **DistListItem** object.


## Remarks

Calling this method sets the  **[IsMarkedAsTask](6dc846e0-c6a6-2151-13e7-f305772b5ae8.md)** property to **False** .
