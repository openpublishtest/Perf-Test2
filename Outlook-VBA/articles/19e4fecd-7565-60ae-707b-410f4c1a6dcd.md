
# ContactItem.ClearTaskFlag Method (Outlook)

Clears the  **[ContactItem](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** object as a task.


## Syntax

 _expression_ . **ClearTaskFlag**

 _expression_ An expression that returns a **ContactItem** object.


## Remarks

Calling this method sets the  **[IsMarkedAsTask](bf651a37-e486-1c54-83d4-3bb3714f7187.md)** property to **False** .
