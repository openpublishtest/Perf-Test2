
# CalendarModule.Name Property (Outlook)

Returns a  **String** value that represents the localized display name for the **[CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** object. Read-only.


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **CalendarModule** object.
