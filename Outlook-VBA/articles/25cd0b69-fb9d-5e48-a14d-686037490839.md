
# AutoFormatRule.Font Property (Outlook)

Returns a  **[ViewFont](cbd7c6ce-f49a-1627-0ad9-a019911fb47b.md)** object that represents the font used to display Outlook items that satisfy the conditions for the formatting rule in the view. Read-only.


## Syntax

 _expression_ . **Font**

 _expression_ A variable that represents an **AutoFormatRule** object.
