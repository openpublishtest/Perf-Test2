
# JournalItem.Conflicts Property (Outlook)

Returns the  **[Conflicts](c4e1c060-519a-a6d1-8fb2-c7dfa1e3e66f.md)** object that represents the items that are in conflict for any Outlook item object. Read-only.


## Syntax

 _expression_ . **Conflicts**

 _expression_ A variable that represents a **JournalItem** object.
