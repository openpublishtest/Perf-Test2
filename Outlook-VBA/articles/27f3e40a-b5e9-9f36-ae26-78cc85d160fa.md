
# Recipient.MeetingResponseStatus Property (Outlook)

Returns an  **[OlResponseStatus](b473d57a-76a1-0862-fecb-baf1cf317772.md)** constant indicating the overall status of the response to the meeting request for the recipient. Read-only.


## Syntax

 _expression_ . **MeetingResponseStatus**

 _expression_ A variable that represents a **Recipient** object.
