
# PostItem.TaskSubject Property (Outlook)

Returns or sets a  **String** value that represents the subject of the task for the **[PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)** object. Read/write.


## Syntax

 _expression_ . **TaskSubject**

 _expression_ An expression that returns a **PostItem** object.


## Remarks

This property returns the value of the  **[Subject](23973924-e5ca-da22-0e63-7b655c96f67b.md)** property if the **[IsMarkedAsTask](a84195b3-8d6e-8842-6747-86977f4dfaad.md)** property is set to **False** .
