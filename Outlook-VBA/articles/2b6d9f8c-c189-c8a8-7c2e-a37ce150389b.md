
# TasksModule.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **TasksModule** object.


## Remarks

This property returns  **olTasksModule** for the **[TasksModule](fc6ae6c9-6b13-b5f2-9506-c3dbbe709df6.md)** object.
