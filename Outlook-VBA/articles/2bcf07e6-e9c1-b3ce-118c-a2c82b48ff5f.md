
# SharingItem.Read Event (Outlook)

Occurs when an instance of the parent object is opened for editing by the user. 


## Syntax

 _expression_ . **Read**

 _expression_ An expression that returns a **SharingItem** object.


## Remarks

The  **Read** event differs from the **[Open](b795dbfa-2d47-0ee4-98ef-0c44bb6a0bec.md)** event in that **Read** occurs whenever the user selects the item in a view that supports in-cell editing as well as when the item is being opened in an **[Inspector](d7384756-669c-0549-1032-c3b864187994.md)** .
