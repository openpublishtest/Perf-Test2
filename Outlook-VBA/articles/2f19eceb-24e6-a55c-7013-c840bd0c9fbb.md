
# CalendarModule.NavigationGroups Property (Outlook)

Returns a  **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object that contains the set of navigation groups associated with the **[CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** object. Read-only.


## Syntax

 _expression_ . **NavigationGroups**

 _expression_ A variable that represents a **CalendarModule** object.
