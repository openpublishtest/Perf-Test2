
# SharingItem.ClearTaskFlag Method (Outlook)

Clears the  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object as a task.


## Syntax

 _expression_ . **ClearTaskFlag**

 _expression_ An expression that returns a **SharingItem** object.


## Remarks

Calling this method sets the  **[IsMarkedAsTask](d71f55d2-d613-d922-dead-14d3b6916a2e.md)** property to **False** .
