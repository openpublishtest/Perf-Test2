
# CalendarView.EndField Property (Outlook)

Returns or sets a  **String** value that represents the name of the property that ends the time duration for Outlook items displayed in the **[CalendarView](37e078b9-9fc6-5894-b043-06d7257666a8.md)** object. Read/write.


## Syntax

 _expression_ . **EndField**

 _expression_ A variable that represents a **CalendarView** object.


## Remarks

The values of the  **[StartField](085c6605-0bff-98a5-fb48-ce32b76037db.md)** and **EndField** properties indicate which Outlook item properties are used by the **CalendarView** object to represent the duration of an Outlook item. Both custom and built-in properties can be specified, but only date/time properties are allowed.
