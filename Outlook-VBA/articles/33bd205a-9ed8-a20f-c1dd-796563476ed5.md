
# OlkListBox.MultiSelect Property (Outlook)

Returns or sets an  **[OlMultiSelect](33d61773-515f-e267-d60a-a3d91f3be9d7.md)** constant that specifies the range of items that can be selected in the list box control. Read/write.


## Syntax

 _expression_ . **MultiSelect**

 _expression_ A variable that represents an **OlkListBox** object.
