
# ContactItem.TaskDueDate Property (Outlook)

Returns or sets a  **Date** value that represents the due date of the task for this **[ContactItem](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** . Read/write.


## Syntax

 _expression_ . **TaskDueDate**

 _expression_ An expression that returns a **ContactItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the **[IsMarkedAsTask](bf651a37-e486-1c54-83d4-3bb3714f7187.md)** property is set to **False** .
