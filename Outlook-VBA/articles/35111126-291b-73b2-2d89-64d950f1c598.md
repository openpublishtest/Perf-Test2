
# JournalItem.Read Event (Outlook)

Occurs when an instance of the parent object is opened for editing by the user. 


## Syntax

 _expression_ . **Read**

 _expression_ A variable that represents a **JournalItem** object.


## Remarks

The  **Read** event differs from the **[Open](d564c47d-00d8-29cf-244c-4a2ef1568e26.md)** event in that **Read** occurs whenever the user selects the item in a view that supports in-cell editing as well as when the item is being opened in an **[Inspector](d7384756-669c-0549-1032-c3b864187994.md)** .
