
# SharingItem.Copy Method (Outlook)

Creates another instance of a  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** .


## Syntax

 _expression_ . **Copy**

 _expression_ A variable that represents a **SharingItem** object.
