
# OlkCheckBox.Value Property (Outlook)

Returns or sets a  **Variant** that specifies the state of the control. Read/write.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents an **OlkCheckBox** object.


## Remarks

For the check box control, this property is  **True** or **False** , or if the **[TripleState](f0f8f928-28e2-3f0f-dcd6-afe0dc4a6618.md)** property is set, the **Value** property can also be **Null** ( **Nothing** in Visual Basic). The default value is **False** .
