
# AddressEntry.Address Property (Outlook)

Returns or sets a  **String** representing the e-mail address of the **[AddressEntry](d4a0a85e-8bab-bc56-57bc-d70c3c570c8e.md)** . Read/write.


## Syntax

 _expression_ . **Address**

 _expression_ A variable that represents an **AddressEntry** object.


## Remarks

The  **Address** property must be set before calling the **[AddressEntry.Details](85457da6-c97a-387d-6c7e-40eb005b25aa.md)** method.
