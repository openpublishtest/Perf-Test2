
# TaskItem.ReminderOverrideDefault Property (Outlook)

Returns or sets a  **Boolean** value that is **True** if the reminder overrides the default reminder behavior for the item. Read/write.


## Syntax

 _expression_ . **ReminderOverrideDefault**

 _expression_ A variable that represents a **TaskItem** object.


## Remarks

You must set the  **ReminderOverrideDefault** property to validate the **[ReminderPlaySound](22698193-bc36-c2fb-3ee1-d04d1e3a15a6.md)** and the **[ReminderSoundFile](29bfa689-08b6-f963-9ecb-3744b1032062.md)** properties.
