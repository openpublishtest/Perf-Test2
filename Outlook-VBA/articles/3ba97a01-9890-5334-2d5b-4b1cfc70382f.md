
# SharingItem.Submitted Property (Outlook)

Returns a  **Boolean** value that is **True** if the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** has been submitted. Read-only.


## Syntax

 _expression_ . **Submitted**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

A message is always created and submitted in a folder, usually the  **Outbox**.
