
# DistListItem.TaskSubject Property (Outlook)

Returns or sets a  **String** value that represents the subject of the task for the **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** object. Read/write.


## Syntax

 _expression_ . **TaskSubject**

 _expression_ An expression that returns a **DistListItem** object.


## Remarks

This property returns the value of the  **[Subject](5c4da614-70dc-e7eb-34d8-963d03637b15.md)** property if the **[IsMarkedAsTask](6dc846e0-c6a6-2151-13e7-f305772b5ae8.md)** property is set to **False** .
