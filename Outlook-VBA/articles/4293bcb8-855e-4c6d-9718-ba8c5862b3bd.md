
# Folder.UserDefinedProperties Property (Outlook)

Returns a  **[UserDefinedProperties](196e5d4c-22be-02d3-95e0-3ea7594c2e4b.md)** object that represents the user-defined custom properties for the **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** object. Read-only.


## Syntax

 _expression_ . **UserDefinedProperties**

 _expression_ A variable that represents a **Folder** object.
