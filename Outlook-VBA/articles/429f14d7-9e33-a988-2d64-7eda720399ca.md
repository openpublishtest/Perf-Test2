
# SharingItem.RequestedFolder Property (Outlook)

Returns an  **[OlDefaultFolders](1a17abd8-09b9-d3e1-2d93-0a4d5580a950.md)** constant that represents the type of default folder to which access is requested by a sharing request. Read-only.


## Syntax

 _expression_ . **RequestedFolder**

 _expression_ An expression that returns a **SharingItem** object.


## Remarks

A  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object that represents a sharing request can only request access to a default folder.
