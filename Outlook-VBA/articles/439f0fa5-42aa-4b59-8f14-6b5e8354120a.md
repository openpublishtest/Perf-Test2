
# Conflict Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](cd319f18-9272-ba9c-6cd3-f8c3f0e9d0f6.md)|Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](dec03590-e169-2ff3-f205-20c347e80693.md)|Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
|[Item](4d2df6de-d68a-fd7a-d41e-90fc13f00ceb.md)|Returns an  **Object** corresponding to the specified Outlook item. Read-only.|
|[Name](46927bd4-a9cc-4098-8089-a40ea968b3ed.md)|Returns a  **String** value that represents the display name for the object. Read-only.|
|[Parent](b8ec7387-0940-3ba8-6012-ca867a796625.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](cd7eaf1e-545b-5a40-d95c-841f72a7a15e.md)|Returns the  **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** object for the current session. Read-only.|
|[Type](6e15fae9-befc-013d-9e95-05c5dc928e09.md)|Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the type of item represented by the **[Conflict](a7c8f12a-08ba-9fff-60b8-a02d1c7f6f33.md)** object. Read-only.|
