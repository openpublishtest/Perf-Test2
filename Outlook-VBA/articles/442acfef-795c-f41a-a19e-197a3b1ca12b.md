
# OlkComboBox.Style Property (Outlook)

Returns or sets an  **[OlComboBoxStyle](8aaeceb7-3928-5691-2f4b-9c27c88ca1c8.md)** constant to specify how the user can choose or set the control's value. Read/write.


## Syntax

 _expression_ . **Style**

 _expression_ A variable that represents an **OlkComboBox** object.


## Remarks

The default value is  **olComboBoxStyleCombo** .
