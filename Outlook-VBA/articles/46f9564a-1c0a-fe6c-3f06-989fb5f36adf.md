
# ExchangeUser.IsUnifiedGroup Method (Outlook)

Determines if the object is a unified group.


## Syntax

 _expression_ . **IsUnifiedGroup**

 _expression_ A variable that represents a **ExchangeUser** object.


### Return Value

Returns the  **Outlook.Folder** object associated with the group using the[GetUnifiedGroup](ec0f58fa-969d-ed38-705b-2c99ccbf3c86.md) and[GetUnifiedGroupFromStore](38a901d3-670f-afd2-a385-3b2bb859cb81.md) methods.
