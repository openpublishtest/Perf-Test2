
# SharingItem.Recipients Property (Outlook)

Returns a  **[Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** collection that represents all the recipients for the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **Recipients**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

A recipient can be specified by a string representing the recipient's display name, alias, or full SMTP e-mail address.
