
# TableView.ShowReadingPane Property (Outlook)

Returns or sets a  **Boolean** value that indicates whether the Reading Pane is displayed in the **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object. Read/write.


## Syntax

 _expression_ . **ShowReadingPane**

 _expression_ A variable that represents a **TableView** object.
