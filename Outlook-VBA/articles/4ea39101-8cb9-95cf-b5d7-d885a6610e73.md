
# MarkAsTaskRuleAction.MarkInterval Property (Outlook)

Returns or sets a constant in the  **[OlMarkInterval](a653146c-8a28-72dd-4ca7-98d8454c6f1f.md)** enumeration that represents the interval before the task is due. Read/write.


## Syntax

 _expression_ . **MarkInterval**

 _expression_ A variable that represents a **MarkAsTaskRuleAction** object.
