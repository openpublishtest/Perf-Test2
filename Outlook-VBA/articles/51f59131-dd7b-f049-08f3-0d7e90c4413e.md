
# NotesModule.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **NotesModule** object.


## Remarks

This property returns  **olNotesModule** for the **[NotesModule](cdbdde08-0773-a78d-3809-a3811975bcc1.md)** object.
