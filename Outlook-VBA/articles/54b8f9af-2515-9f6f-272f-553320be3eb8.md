
# OlkCheckBox.Change Event (Outlook)

Occurs when the  **[Value](3872331d-9179-3412-1512-bc8ed5d35beb.md)** property changes.


## Syntax

 _expression_ . **Change**

 _expression_ A variable that represents an **OlkCheckBox** object.
