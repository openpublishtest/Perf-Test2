
# TimeZones.Item Method (Outlook)

Returns a  **[TimeZone](b27da70d-e545-cc13-9529-cfd327ab7a7c.md)** object from the collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that represents a **TimeZones** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|An  **Integer** representing a one-based index into the **[TimeZones](c68f8589-44e9-3c12-45c1-96943fa9bcb7.md)** collection.|

### Return Value

A  **TimeZone** object that represents the specified object in the collection.
