
# OlSortOrder Enumeration (Outlook)

Indicates the sort order.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **olAscending**|1|Ascending order|
| **olDescending**|2|Descending order|
| **olSortNone**|0|Not sorted|

## Remarks

Used as an optional parameter to the [AddressEntries.Sort Method (Outlook)](9b381837-9fe9-1041-8297-e8c8dbcdc2e4.md) to specify the order that address entries should be sorted by the _Property_ parameter.
