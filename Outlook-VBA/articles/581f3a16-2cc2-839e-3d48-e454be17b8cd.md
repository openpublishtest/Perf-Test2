
# DistListItem.Read Event (Outlook)

Occurs when an instance of the parent object is opened for editing by the user. 


## Syntax

 _expression_ . **Read**

 _expression_ A variable that represents a **DistListItem** object.


## Remarks

The  **Read** event differs from the **[Open](5bc620d5-9dd5-e7a0-10ff-a1ea7e2fdc56.md)** event in that **Read** occurs whenever the user selects the item in a view that supports in-cell editing as well as when the item is being opened in an **[Inspector](d7384756-669c-0549-1032-c3b864187994.md)** .
