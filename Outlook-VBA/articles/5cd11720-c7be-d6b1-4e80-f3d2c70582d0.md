
# AutoFormatRules.Count Property (Outlook)

Returns a  **Long** value indicating the count of **[AutoFormatRule](6d295c41-17f9-8e67-4595-4330fd3cec99.md)** objects in the specified collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents an **AutoFormatRules** object.
