
# Explorer.PreviewPane Property (Outlook)

The [PreviewPane](fd4f497b-7085-6e0f-018b-17845f4dfe61.md) object displays content in a "single pane mode" by showing only the Preview Pane view.


## Syntax

 _expression_ . **PreviewPane**

 _expression_ A variable that represents an **Explorer** object.
