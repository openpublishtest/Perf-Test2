
# Rules Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](406b1f7c-1714-5f0e-5d9f-37ddc963ca69.md)|Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent Outlook application for the object. Read-only.|
