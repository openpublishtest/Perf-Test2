
# NavigationGroups.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **NavigationGroups** object.


## Remarks

This property returns  **olNavigationGroups** for the **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object.
