
# Action.ShowOn Property (Outlook)

Returns or sets an  **[OlActionShowOn](6a6e4156-d593-b5c7-8ed1-e133d61332df.md)** constant representing the location where the action will be shown. Read/write.


## Syntax

 _expression_ . **ShowOn**

 _expression_ A variable that represents an **Action** object.
