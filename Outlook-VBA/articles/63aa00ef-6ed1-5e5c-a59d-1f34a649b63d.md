
# SharingItem.ConversationIndex Property (Outlook)

Returns a  **String** representing the index of the conversation thread of the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **ConversationIndex**

 _expression_ A variable that represents a **SharingItem** object.
