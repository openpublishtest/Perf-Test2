
# SelectNamesDialog.Class Property (Outlook)

Returns a constant in the  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** enumeration indicating the class of the **[SelectNamesDialog](1522736a-3cad-9f1c-4da9-b52a3a01731c.md)** object. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **SelectNamesDialog** object.
