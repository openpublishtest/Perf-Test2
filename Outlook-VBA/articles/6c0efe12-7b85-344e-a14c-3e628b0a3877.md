
# OlkTextBox.Value Property (Outlook)

Returns or sets a  **Variant** that represents the content of the control. Read/write.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents an **OlkTextBox** object.


## Remarks

The value is the same as the value of  **[Text](1dc8c0ec-00d1-5066-f204-68f7e6b30a42.md)** , except that this property is **Variant** and **Text** is strongly typed as **String** .
