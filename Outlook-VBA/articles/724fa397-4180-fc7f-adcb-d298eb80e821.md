
# JournalModule.NavigationGroups Property (Outlook)

Returns a  **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object that contains the set of navigation groups associated with the **[JournalModule](5a696d10-8a10-c01d-cf65-f8a65718f120.md)** object. Read-only.


## Syntax

 _expression_ . **NavigationGroups**

 _expression_ A variable that represents a **JournalModule** object.
