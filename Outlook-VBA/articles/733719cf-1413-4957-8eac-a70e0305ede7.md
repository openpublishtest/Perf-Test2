
# SharingItem.ItemProperties Property (Outlook)

Returns an  **[ItemProperties](34a110ed-6617-72da-1e98-a9773c705b40.md)** collection that represents all standard and user-defined properties associated with the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **ItemProperties**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

The  **ItemProperties** collection is a zero-based collection, meaning that the first object in the collection is referenced by the index 0.
