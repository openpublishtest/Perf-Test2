
# Columns.Parent Property (Outlook)

Returns the parent object of the  **[Columns](628bf0cf-4ee8-5e5c-09d7-89d7adf256ca.md)** object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **Columns** object.
