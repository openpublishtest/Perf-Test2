
# TaskItem.Ownership Property (Outlook)

Returns an  **[OlTaskOwnership](1b3f1b21-e928-4cc2-e6ae-81cf9db688ad.md)** specifying the ownership state of the task. Read-only.


## Syntax

 _expression_ . **Ownership**

 _expression_ A variable that represents a **TaskItem** object.
