
# MailItem.ClearTaskFlag Method (Outlook)

Clears the  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** object as a task.


## Syntax

 _expression_ . **ClearTaskFlag**

 _expression_ An expression that returns a **MailItem** object.


## Remarks

Calling this method sets the  **[IsMarkedAsTask](6cc4530d-fa74-916b-654d-db995d9a989f.md)** property to **False** .
