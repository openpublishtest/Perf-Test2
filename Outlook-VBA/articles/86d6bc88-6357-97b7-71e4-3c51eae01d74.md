
# Account.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **Account** object.


## Remarks

The parent of the  **[Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** object is the **[Accounts](2510b7d7-5062-8ea3-dda4-b544d2882a2b.md)** object.
