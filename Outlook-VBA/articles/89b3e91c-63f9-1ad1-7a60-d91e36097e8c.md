
# SharingItem.SaveSentMessageFolder Property (Outlook)

Returns or sets a  **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** object that represents the folder in which a copy of the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** will be saved after being sent. Read/write.


## Syntax

 _expression_ . **SaveSentMessageFolder**

 _expression_ A variable that represents a **SharingItem** object.
