
# Accounts.Item Method (Outlook)

Returns an  **[Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** object specified by _Index_ .


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ A variable that represents an **Accounts** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|A one-based  **Long** that indexes into the **[Accounts](2510b7d7-5062-8ea3-dda4-b544d2882a2b.md)** collection, or a **String** that specifies the **[DisplayName](20fd9286-c7d9-3bb3-5b33-137313f1c8d5.md)** of an **Account** .|

### Return Value

An  **Account** object that matches the account specified by _Index_ .
