
# TaskItem.ResponseState Property (Outlook)

Returns an  **[OlTaskResponse](7616cbdc-fc9c-abbe-fd07-ebdadc13ede2.md)** constant indicating the overall status of the response to the specified task request. Read-only.


## Syntax

 _expression_ . **ResponseState**

 _expression_ A variable that represents a **TaskItem** object.
