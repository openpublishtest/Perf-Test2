
# AppointmentItem.IsRecurring Property (Outlook)

Returns a  **Boolean** value that is **True** if the appointment is a recurring appointment. Read-only.


## Syntax

 _expression_ . **IsRecurring**

 _expression_ A variable that represents an **AppointmentItem** object.


## Remarks

When the  **[GetRecurrencePattern](a9f67c5b-a77f-4e34-e654-d12560a6dba0.md)** method is used with an **[AppointmentItem](204a409d-654e-27aa-643a-8344c631b82d.md)** object, this property is set to **True** .
