
# SharingItem.TaskStartDate Property (Outlook)

Returns or sets a  **Date** value that represents the start date of the task for this **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object. Read/write.


## Syntax

 _expression_ . **TaskStartDate**

 _expression_ An expression that returns a **SharingItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the **[IsMarkedAsTask](d71f55d2-d613-d922-dead-14d3b6916a2e.md)** property is set to **False** .
