
# OlkTextBox.DragBehavior Property (Outlook)

Returns or sets an  **[OlDragBehavior Enumeration](1e8c29d4-7800-663f-fb5f-aebc2a6b89fe.md)** constant that indicates whether the system enables the drag-and-drop feature for this control. Read/write.


## Syntax

 _expression_ . **DragBehavior**

 _expression_ A variable that represents an **OlkTextBox** object.


## Remarks

The default value is  **olDragDropDisabled** .
