
# AppointmentItem.Resources Property (Outlook)

Returns a semicolon-delimited  **String** of resource names for the meeting. Read/write.


## Syntax

 _expression_ . **Resources**

 _expression_ A variable that represents an **AppointmentItem** object.


## Remarks

This property contains the display names only. The  **[Recipients](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** collection should be used to modify the resource recipients. Resources are added as **[BCC](6454f9b1-1bfa-d4d4-ca95-7a19db920977.md)** recipients to the collection.
