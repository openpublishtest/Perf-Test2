
# SharingItem.BillingInformation Property (Outlook)

Returns or sets a  **String** representing the billing information associated with the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read/write.


## Syntax

 _expression_ . **BillingInformation**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

This is a free-form text field.
