
# ExchangeDistributionList.IsUnifiedGroup Method (Outlook)

Determines if the object is a unified group.


## Syntax

 _expression_ . **IsUnifiedGroup**

 _expression_ A variable that represents a **ExchangeDistributionList** object.


### Return Value

Returns the  **Outlook.Folder** object associated with the group using the[GetUnifiedGroup](9b129256-02c0-438a-9098-c0925ec60388.md) and[GetUnifiedGroupFromStore](8565a110-d9d9-bc58-a5c8-a0ac9da8ec0c.md) methods.
