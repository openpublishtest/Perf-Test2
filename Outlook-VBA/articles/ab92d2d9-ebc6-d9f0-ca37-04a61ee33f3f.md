
# PreviewPane.Parent Property (Outlook)

Returns the parent property for the [PreviewPane](fd4f497b-7085-6e0f-018b-17845f4dfe61.md) Object. Read only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **PreviewPane** object.
