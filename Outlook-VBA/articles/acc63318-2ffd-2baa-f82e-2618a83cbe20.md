
# PeopleView.Class Property (Outlook)
Returns an [OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md) constant indicating the object's class. Read-only.

## Version information

Version Added: Outlook 2013 


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **PeopleView** object.


## Property value

 **OLOBJECTCLASS**
