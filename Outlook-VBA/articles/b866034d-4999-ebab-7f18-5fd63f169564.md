
# OrderFields.Insert Method (Outlook)

Creates a new  **[OrderField](4ae32270-bde9-3178-bca3-f8d145779d3d.md)** object and inserts it at the specified index within the **[OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** collection.


## Syntax

 _expression_ . **Insert**( **_PropertyName_** , **_Index_** , **_IsDescending_** )

 _expression_ A variable that represents an **OrderFields** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _PropertyName_|Required| **String**|The name of the property to which the new object is associated.|
| _Index_|Required| **Variant**|Either the index number at which to insert the new object, or a value used to match the  **[ViewXMLSchemaName](a88c22ff-3d30-a4f2-87f6-6c32c1c2acb7.md)** property value of an object in the collection at where the new object is to be inserted.|
| _IsDescending_|Optional| **Boolean**|The value used to set the  **[IsDescending](941f7144-748a-7b57-35f1-3e29077b926d.md)** property of the new **OrderField** object. If this value is not specified, the default value of the **IsDescending** property is used.|

### Return Value

An  **OrderField** object that represents the new order field.
