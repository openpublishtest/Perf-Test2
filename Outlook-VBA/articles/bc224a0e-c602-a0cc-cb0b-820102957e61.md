
# SharingItem.UserProperties Property (Outlook)

Returns the  **[UserProperties](20b49c86-d74f-9bda-382c-559af278c148.md)** collection that represents all the user properties for the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **UserProperties**

 _expression_ A variable that represents a **SharingItem** object.
