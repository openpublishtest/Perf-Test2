
# TableView.ViewFields Property (Outlook)

Returns a  **[ViewFields](2516faed-ed11-6cb3-ce9c-b6afa788e909.md)** object that represents the set of fields with which Outlook items are displayed in the **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object. Read-only.


## Syntax

 _expression_ . **ViewFields**

 _expression_ A variable that represents a **TableView** object.
