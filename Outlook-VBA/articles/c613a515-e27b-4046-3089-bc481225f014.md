
# OlkControl.HorizontalLayout Property (Outlook)

Returns or sets an  **[OlHorizontalLayout](97d132c7-658a-28d4-b0e4-4352e5341987.md)** constant that specifies any alignment in the horizontal direction of this control with other selected controls on the form, or whether the control is adjusted in the horizontal direction automatically when the form's width is changed. Read/write.


## Syntax

 _expression_ . **HorizontalLayout**

 _expression_ A variable that represents an **OlkControl** object.
