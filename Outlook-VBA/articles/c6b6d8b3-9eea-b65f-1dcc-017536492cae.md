
# PostItem.ClearTaskFlag Method (Outlook)

Clears the  **[PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)** object as a task.


## Syntax

 _expression_ . **ClearTaskMark**

 _expression_ An expression that returns a **PostItem** object.


## Remarks

Calling this method sets the  **[IsMarkedAsTask](a84195b3-8d6e-8842-6747-86977f4dfaad.md)** property to **False** .
