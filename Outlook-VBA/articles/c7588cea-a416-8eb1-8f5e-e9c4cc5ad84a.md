
# SharingItem.ReminderPlaySound Property (Outlook)

Returns or sets a  **Boolean** value that is **True** if the reminder should play a sound when it occurs for the specified **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read/write.


## Syntax

 _expression_ . **ReminderPlaySound**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

The  **ReminderPlaySound** property must be set in order to validate the **[ReminderSoundFile](55974c0e-3b72-b64e-87ed-53ffc280721b.md)** property.
