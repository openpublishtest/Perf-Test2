
# Columns.Application Property (Outlook)

Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent application (Outlook) for the **[Columns](628bf0cf-4ee8-5e5c-09d7-89d7adf256ca.md)** object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **Columns** object.
