
# TasksModule.NavigationGroups Property (Outlook)

Returns a  **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** object that contains the set of navigation groups associated with the **[TasksModule](fc6ae6c9-6b13-b5f2-9506-c3dbbe709df6.md)** object. Read-only.


## Syntax

 _expression_ . **NavigationGroups**

 _expression_ A variable that represents a **TasksModule** object.
