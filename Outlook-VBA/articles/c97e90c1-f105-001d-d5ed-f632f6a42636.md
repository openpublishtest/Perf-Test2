
# OlkTextBox.Change Event (Outlook)

Occurs when the  **[Value](6c0efe12-7b85-344e-a14c-3e628b0a3877.md)** property changes.


## Syntax

 _expression_ . **Change**

 _expression_ A variable that represents an **OlkTextBox** object.
