
# SharingItem.ReplyRecipientNames Property (Outlook)

Returns a semicolon-delimited  **String** list of reply recipients for the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **ReplyRecipientNames**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

This property only contains the display names for the reply recipients. The reply recipients list should be set by using the  **[ReplyRecipients](6c1a49bd-899b-93f1-f8b9-64051fd16c35.md)** collection.
