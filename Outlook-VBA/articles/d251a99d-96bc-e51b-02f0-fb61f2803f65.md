
# Conversation.Application Property (Outlook)

Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent Microsoft Outlook application for the **[Conversation](2705d38a-ebc0-e5a7-208b-ffe1f5446b1b.md)** object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **Conversation** object.
