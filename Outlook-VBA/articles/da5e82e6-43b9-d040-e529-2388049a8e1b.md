
# DocumentItem.Read Event (Outlook)

Occurs when an instance of the parent object is opened for editing by the user. 


## Syntax

 _expression_ . **Read**

 _expression_ A variable that represents a **DocumentItem** object.


## Remarks

The  **Read** event differs from the **[Open](e7d95148-9fa2-3f0f-cbfc-f835c9017c3b.md)** event in that **Read** occurs whenever the user selects the item in a view that supports in-cell editing as well as when the item is being opened in an **[Inspector](d7384756-669c-0549-1032-c3b864187994.md)** .
