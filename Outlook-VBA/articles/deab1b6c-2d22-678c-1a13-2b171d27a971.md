
# SharingItem.MarkAsTask Method (Outlook)

Marks a  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object as a task and assigns a task interval for the object.


## Syntax

 _expression_ . **MarkAsTask**( **_MarkInterval_** )

 _expression_ An expression that returns a **SharingItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _MarkInterval_|Required| **[OlMarkInterval](a653146c-8a28-72dd-4ca7-98d8454c6f1f.md)**|The task interval for the  **SharingItem** .|

## Remarks

Calling this method sets the  **[IsMarkedAsTask](d71f55d2-d613-d922-dead-14d3b6916a2e.md)** property to **True** and updates the **[TaskStartDate](93fedab7-999e-4ee0-e65e-57d64ae02167.md)** , **[TaskDueDate](baf6c9c8-cdf3-2e8b-71da-10a393421f56.md)** , and **[TaskOrdinal](4164fa78-c0cf-e359-2707-025d6d49f145.md)** properties depending on the value provided in _MarkInterval_.
