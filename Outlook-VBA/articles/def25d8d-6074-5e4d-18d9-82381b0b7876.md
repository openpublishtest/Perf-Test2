
# ContactItem.MarkAsTask Method (Outlook)

Marks a  **[ContactItem](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** object as a task and assigns a task interval for the object.


## Syntax

 _expression_ . **MarkAsTask**( **_MarkInterval_** )

 _expression_ An expression that returns a **ContactItem** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _MarkInterval_|Required| **[OlMarkInterval](a653146c-8a28-72dd-4ca7-98d8454c6f1f.md)**|The task interval for the  **ContactItem** .|

## Remarks

Calling this method sets the value of several other properties, depending on the value provided in  _MarkInterval_. For more information about the properties set by specifying  _MarkInterval_, see [OlMarkInterval Enumeration](a653146c-8a28-72dd-4ca7-98d8454c6f1f.md).
