
# Explorer.Close Method (Outlook)

Closes the  **[Explorer](026591e5-049f-503a-4166-34e6dbc225fb.md)** object.


## Syntax

 _expression_ . **Close**()

 _expression_ A variable that represents an **Explorer** object.
