
# SharingItem.Conflicts Property (Outlook)

Return the  **[Conflicts](c4e1c060-519a-a6d1-8fb2-c7dfa1e3e66f.md)** object that represents the items that are in conflict with the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **Conflicts**

 _expression_ A variable that represents a **SharingItem** object.
