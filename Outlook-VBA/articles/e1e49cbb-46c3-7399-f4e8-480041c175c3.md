
# PeopleView.Copy Method (Outlook)
Creates a new [View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md) object based on the existing[PeopleView](7b569709-5da8-a950-a0fb-9d64b520a21b.md) object.

## Version information

Version Added: Outlook 2013 


## Syntax

 _expression_ . **Copy**_(Name,_ _SaveOption)_

 _expression_ A variable that represents a **PeopleView** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|||||
| _Name_|Required| **String**|The name of the new view.|
| _SaveOption_|Optional|[OlViewSaveOption](c08bab4d-ecdd-a2ac-1cdc-fa910f9585e0.md)|The save option for the new view.|
