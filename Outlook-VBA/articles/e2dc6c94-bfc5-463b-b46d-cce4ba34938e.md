
# Views Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|

|[Parent](3ada0c2f-3719-f7a9-641c-7d0d7174ca37.md)|Returns the parent  **Object** of the specified object. Read-only.|
