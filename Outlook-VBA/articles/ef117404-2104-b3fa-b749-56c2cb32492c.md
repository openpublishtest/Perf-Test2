
# Views Members (Outlook)
Contains a collection of all  **[View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md)** objects in the current folder.

Contains a collection of all  **[View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md)** objects in the current folder.


## Events



|**Name**|**Description**|
|:-----|:-----|
|[ViewAdd](926eb4eb-7585-5bb0-b214-6e116a01375e.md)|Occurs when a view is added to the collection. Microsoft Outlook creates the new view and passes it to this event.|
|[ViewRemove](a0d405fd-aa57-c333-8e33-aa482019d9c8.md)|Occurs when a view has been removed from the specified collection.|

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](8005ca2e-8b28-1286-74d1-448f2a168c65.md)|Creates a new view in the  **[Views](5dd7edc2-12a2-f4c2-d158-8053d80e8dc9.md)** collection.|
|[Item](c8fa2aec-5e38-4233-bf58-ec8669377ec7.md)|Returns an  **[View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md)** object from the collection.|
|[Remove](73a92be6-8dc4-6fb9-7f20-0ff678445737.md)|Removes an object from the collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](06912bad-7d59-ea9b-0491-463386474d99.md)|Returns an  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](2ffe97b5-8ccb-8d13-b35f-0cc0399fa719.md)|Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.|
|[Count](cbdd4b27-6383-0df9-2b8a-715fc40f1eb6.md)|Returns a  **Long** indicating the count of objects in the specified collection. Read-only.|
|[Parent](3ada0c2f-3719-f7a9-641c-7d0d7174ca37.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](677d7b97-b138-3506-7b45-26d091f9ba6e.md)|Returns the  **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** object for the current session. Read-only.|
