
# SharingItem.Importance Property (Outlook)

Returns or sets an  **[OlImportance](71e04f9a-fab6-153f-b046-11f7ec50e8e4.md)** constant indicating the relative importance level for the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read/write.


## Syntax

 _expression_ . **Importance**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

This property corresponds to the MAPI property  **PidTagImportance** .
