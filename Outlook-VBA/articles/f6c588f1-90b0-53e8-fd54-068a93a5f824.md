
# Inspector.IsWordMail Method (Outlook)

Determines whether the mail message associated with an inspector is displayed in an Outlook  **[Inspector](d7384756-669c-0549-1032-c3b864187994.md)** or in Microsoft Word.


## Syntax

 _expression_ . **IsWordMail**

 _expression_ A variable that represents an **Inspector** object.


### Return Value

 **True** if the mail message is displayed in Microsoft Word (that is, if Word Mail is in use); otherwise, **False** .
