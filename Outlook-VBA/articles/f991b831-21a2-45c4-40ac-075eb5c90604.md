
# MailModule.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **MailModule** object.


## Remarks

This property returns  **olMailModule** for the **[MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** object.
