
# SharingItem.Class Property (Outlook)

Returns an  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** constant indicating the object's class. Read-only.


## Syntax

 _expression_ . **Class**

 _expression_ A variable that represents a **SharingItem** object.


## Remarks

This property always returns  **olSharing** for the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** object.
