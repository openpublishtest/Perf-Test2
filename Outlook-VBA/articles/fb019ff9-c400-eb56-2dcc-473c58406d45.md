
# DistListItem.TaskDueDate Property (Outlook)

Returns or sets a  **Date** value that represents the due date of the task for this **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** . Read/write.


## Syntax

 _expression_ . **TaskDueDate**

 _expression_ An expression that returns a **DistListItem** object.


## Remarks

This property returns  **Null** ( **Nothing** in Visual Basic) if the **[IsMarkedAsTask](6dc846e0-c6a6-2151-13e7-f305772b5ae8.md)** property is set to **False** .
