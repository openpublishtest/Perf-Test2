
# LegendKey.PictureUnit2 Property (PowerPoint)

Returns or sets the unit for each picture on the chart if the  **[PictureType](9f605c49-eb0f-2e04-304a-34fbf15143b1.md)** property is set to **xlStackScale**; otherwise, this property is ignored. Read/write **Double**.


## Syntax

 _expression_. **PictureUnit2**

 _expression_ A variable that represents a **[LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.
