
# NamedSlideShows Members (PowerPoint)
A collection of all the  **[NamedSlideShow](2f5ddeeb-ecf5-50da-99b9-b38e789fd5bb.md)** objects in the presentation. Each **NamedSlideShow** object represents a custom slide show.

|:-----|:-----|
