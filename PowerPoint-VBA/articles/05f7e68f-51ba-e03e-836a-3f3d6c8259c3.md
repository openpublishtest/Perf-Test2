
# UpBars.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents an **[UpBars](8a176f01-01a6-86bc-a69b-29763ebb1481.md)** object.
