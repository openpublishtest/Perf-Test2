
# CustomLayout.Select Method (PowerPoint)

Selects the specified  **[CustomLayout](67829704-0314-aed2-5415-6736cefc197e.md)** object.


## Syntax

 _expression_. **Select**

 _expression_ A variable that represents a **CustomLayout** object.
