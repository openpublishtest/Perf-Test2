
# Series.ClearFormats Method (PowerPoint)

Clears the formatting of the object.


## Syntax

 _expression_. **ClearFormats**

 _expression_ A variable that represents a **[Series](5c8c2d92-d8ca-4d21-e213-c374292275d4.md)** object.
