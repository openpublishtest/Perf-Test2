
# SlideRange.Layout Property (PowerPoint)

Returns or sets a  **PpSlideLayout** constant that represents the slide layout. Read/write.


## Syntax

 _expression_. **Layout**

 _expression_ A variable that represents a **SlideRange** object.


## Remarks
