
# CellRange Members (PowerPoint)
A collection of  **[Cell](e89e5d69-33b1-d7b1-0a6c-4dfd8b676977.md)** objects in a table column or row. The **CellRange** collection represents all the cells in the specified column or row. To use the **CellRange** collection, use the **Cells** keyword.
