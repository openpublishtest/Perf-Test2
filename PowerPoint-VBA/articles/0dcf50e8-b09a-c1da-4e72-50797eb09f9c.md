
# Slide.CustomLayout Property (PowerPoint)

Returns a  **[CustomLayout](67829704-0314-aed2-5415-6736cefc197e.md)** object that represents the custom layout associated with the specified slide. Read-only.


## Syntax

 _expression_. **CustomLayout**

 _expression_ A variable that represents a **Slide** object.


### Return Value

CustomLayout
