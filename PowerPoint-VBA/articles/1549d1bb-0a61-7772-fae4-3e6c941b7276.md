
# Series.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Series](5c8c2d92-d8ca-4d21-e213-c374292275d4.md)** object.
