
# ResampleMediaTask.Shape Property (PowerPoint)

Returns the shape that is associated with the specified media asset. Read-only.


## Syntax

 _expression_. **Shape**

 _expression_ A variable that represents a **ResampleMediaTask** object.


### Return Value

Shape
