
# Points.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Points](f3ee69d3-ab8f-e300-bbf4-00ea97d47c2a.md)** object.
