
# Points.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **[Points](f3ee69d3-ab8f-e300-bbf4-00ea97d47c2a.md)** object.
