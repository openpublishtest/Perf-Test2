
# ChartGroup.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartGroup](5caa5855-bd69-3fbc-f601-504e431a42e9.md)** object.
