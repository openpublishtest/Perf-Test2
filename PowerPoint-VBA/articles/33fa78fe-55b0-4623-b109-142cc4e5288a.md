
# ChartArea Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](7444fd0d-4249-33c5-263d-8e095ad908b4.md)|When used without an object qualifier, returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](27116510-d18e-a0a6-c406-54c8bf73652a.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](852ef3ee-9aa3-4c19-cc39-d6a4a2995a33.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.|
|[Height](01deb7e5-af03-df99-5a3b-15cfd5236f78.md)|Returns or sets the height, in points, of the object. Read/write  **Double**.|
|[Left](49dc752d-ebfb-318b-2ac5-7d72146593dc.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double**.|
|[Name](a901e04d-cd01-b08e-6f5a-ac6bb7e9b893.md)|Returns the name of the object. Read-only  **String**.|
|[Parent](6e720423-30f2-dc6d-278d-329eb2938531.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Shadow](03f26807-98c4-4436-5fd7-1dba91f9aec6.md)|Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.|
|[Top](dac1cfbb-9fc3-c4ce-205a-bcaad54c5b88.md)|Returns or sets the distance, in points, from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double**.|
|[Width](140734ad-cf7f-1324-caae-7138de074e97.md)|Returns or sets the width, in points, of the object. Read/write  **Double**.|
