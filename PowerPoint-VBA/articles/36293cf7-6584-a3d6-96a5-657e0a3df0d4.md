
# MediaFormat.Application Property (PowerPoint)

Returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the creator of the specified object.


## Syntax

 _expression_. **Application**

 _expression_ An expression that returns a **MediaFormat** object.


### Return Value

Application
