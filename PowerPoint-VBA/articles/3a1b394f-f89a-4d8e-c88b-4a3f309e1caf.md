
# Axes.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **[Axes](71f1e1fc-7086-a84e-1e05-6fa50597b49b.md)** object.
