
# Trendline.Border Property (PowerPoint)

Returns the border of the object. Read-only  **[ChartBorder](fd651a9a-4068-9a9b-f605-9228da5e6183.md)**.


## Syntax

 _expression_. **Border**

 _expression_ A variable that represents a **[Trendline](74755c19-0a7d-cbbf-857e-78740adf6aa4.md)** object.
