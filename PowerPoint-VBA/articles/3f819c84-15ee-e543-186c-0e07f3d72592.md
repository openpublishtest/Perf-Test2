
# DataLabel.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[DataLabel](a17d23c5-0361-9129-28e5-b892f6966bda.md)** object.
