
# MediaFormat.SampleWidth Property (PowerPoint)

Returns the resolution width of the media sample. Read-only.


## Syntax

 _expression_. **SampleWidth**

 _expression_ An expression that returns a **MediaFormat** object.


### Return Value

Long
