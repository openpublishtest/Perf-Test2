
# DataLabel.NumberFormatLocal Property (PowerPoint)

Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.


## Syntax

 _expression_. **NumberFormatLocal**

 _expression_ A variable that represents a **[DataLabel](a17d23c5-0361-9129-28e5-b892f6966bda.md)** object.


## Remarks

The  **Format** function uses different format code strings than do the **[NumberFormat](b19a87fb-3a20-dc56-2bb0-1e606e9a765c.md)** and **NumberFormatLocal** properties.
