
# TickLabels.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[TickLabels](2ba878bf-3a76-1350-2bd4-615c2520f042.md)** object.
