
# FileConverters.Item Method (PowerPoint)

Returns an individual  **FileConverter** object in a collection.


## Syntax

 _expression_. **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[FileConverters](b175c08d-a910-636e-cdbe-d4f0e6af595e.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

FileConverter
