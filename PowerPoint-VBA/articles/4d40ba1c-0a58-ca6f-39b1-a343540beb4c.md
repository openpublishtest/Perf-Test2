
# CategoryCollection Members (PowerPoint)
Represents the collection of visible chart categories in the presentation.

Represents the collection of visible chart categories in the presentation.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Item](b1da03a5-89cd-2cfb-bee9-f83aaf30435f.md)|Returns an individual chart category.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](927c3780-923b-bc60-0e48-814a1806a69d.md)|When used without an object qualifier, returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Count](7052c3a5-7ef8-9ba5-c641-748c79558903.md)|Returns a  **Long** that represents the number of categories in the collection. Read-only.|
|[Creator](79a34381-f75d-12d8-91a0-02c2db0f3b34.md)|Returns a  **Long** that represents the four-character creator code for the application in which the specified object was created. For example, if the object was created in PowerPoint, this property returns the hexadecimal number 50575054. Read-only.|
|[Parent](79600e7d-5c35-f995-21c9-44a9367e9369.md)|Returns the parent for the specified object. Read-only  **Object**.|
