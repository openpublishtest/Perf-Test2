
# ChartGroups.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartGroups](2db874db-91af-0b1e-7496-92a8443caade.md)** object.
