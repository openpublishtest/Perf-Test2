
# ChartFormat.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)** object.
