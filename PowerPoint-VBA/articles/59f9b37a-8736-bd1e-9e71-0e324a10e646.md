
# ChartGroup.BinsOverflowValue Property (PowerPoint)

If an [ChartGroup.BinsOverflowEnabled Property (PowerPoint)](9d5e5296-b80c-f6dd-b418-1d0cd3a9adce.md) is **True**, specifies the value above which an overflow bin is displayed. Read/write  **Double**.


## Syntax

 _expression_. **BinsOverflowValue**

 _expression_ A variable that represents a **ChartGroup** object.
