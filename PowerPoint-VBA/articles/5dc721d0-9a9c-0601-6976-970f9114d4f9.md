
# DataTable.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[DataTable](eaa7cdda-e374-7d19-47a6-87e4458fc244.md)** object.
