
# Point.Delete Method (PowerPoint)

Deletes the object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **[Point](e0137fdd-5632-88d7-a6c0-57a76717e736.md)** object.


## Remarks

Deleting a  **Point** object deletes the entire series.
