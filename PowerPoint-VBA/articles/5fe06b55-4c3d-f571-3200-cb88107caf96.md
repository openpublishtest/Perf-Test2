
# Legend.Width Property (PowerPoint)

Returns or sets the width, in points, of the object. Read/write  **Double**.


## Syntax

 _expression_. **Width**

 _expression_ A variable that represents a **[Legend](7be25694-8694-049a-c31f-533fe6fd0562.md)** object.
