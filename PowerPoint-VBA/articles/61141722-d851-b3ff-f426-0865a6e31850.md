
# CustomLayout.Background Property (PowerPoint)

Returns a  **[ShapeRange](0a194183-380e-ffb6-9336-b5bd311e917d.md)** object that represents the slide background. Read-only.


## Syntax

 _expression_. **Background**

 _expression_ A variable that represents a **CustomLayout** object.


### Return Value

ShapeRange
