
# ChartBorder.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartBorder](fd651a9a-4068-9a9b-f605-9228da5e6183.md)** object.
