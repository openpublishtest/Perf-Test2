
# Series.ParentDataLabelOption Property (PowerPoint)

Specifies the parent data label option (banner, overlapping, or none) for the specified series within the chart group. Read/write [XlParentDataLabelOptions Enumeration (PowerPoint)](566194d6-f4e3-53af-723c-025bf3909578.md).


## Syntax

 _expression_. **ParentDataLabelOption**

 _expression_ A variable that represents a **Series** object.
