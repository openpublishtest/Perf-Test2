
# SeriesLines.Border Property (PowerPoint)

Returns the border of the object. Read-only  **[ChartBorder](fd651a9a-4068-9a9b-f605-9228da5e6183.md)**.


## Syntax

 _expression_. **Border**

 _expression_ A variable that represents a **[SeriesLines](5d953ed4-ca16-3cb3-ba8f-1742e4a56cb6.md)** object.
