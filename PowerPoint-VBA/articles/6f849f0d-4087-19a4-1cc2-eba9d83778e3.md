
# DataLabels.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[DataLabels](a0d0b0ec-6a12-9a5c-1026-1e1d85e488fa.md)** object.
