
# Floor.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[Floor](ed9ff3d1-8001-840c-d26e-7513ebe73ae9.md)** object.
