
# Gridlines.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[GridLines](10b45c4c-05a3-f722-15ca-ad0242625edb.md)** object.
