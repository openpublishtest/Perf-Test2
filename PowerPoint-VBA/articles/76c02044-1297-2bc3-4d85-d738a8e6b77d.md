
# FileConverter.Name Property (PowerPoint)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[FileConverter](6baf5bd8-6644-0784-a049-96c3d733043f.md)** object.
