
# AxisTitle.Position Property (PowerPoint)

Returns or sets the position of the axis title on the chart. Read/write  **[XlChartElementPosition](9d383293-0cf7-d3ec-28c9-97b100546fec.md)**.


## Syntax

 _expression_. **Position**

 _expression_ A variable that represents an **[AxisTitle](8eddc95c-2353-43fa-c055-ee76de28009d.md)** object.
