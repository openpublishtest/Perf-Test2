
# ChartFormat.ThreeD Property (PowerPoint)

Returns the 3-D-effect formatting properties for the specified chart. Read-only  **[ThreeDFormat](d6eb7b36-57df-727e-fc5b-50b8c4790c1c.md)**.


## Syntax

 _expression_. **ThreeD**

 _expression_ A variable that represents a **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)** object.
