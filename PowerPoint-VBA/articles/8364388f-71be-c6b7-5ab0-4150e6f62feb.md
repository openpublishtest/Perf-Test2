
# Master.CustomLayouts Property (PowerPoint)

Returns a  **[CustomLayouts](9ce682fb-545c-55cb-e9ac-3475f7556af1.md)** object that represents the custom layouts associated with the presentation design of the specified **[Master](22e8805e-6469-1a34-7f7b-f1ea5c6c49ff.md)** object. Read-only.


## Syntax

 _expression_. **CustomLayouts**

 _expression_ A variable that represents a **Master** object.


### Return Value

CustomLayouts
