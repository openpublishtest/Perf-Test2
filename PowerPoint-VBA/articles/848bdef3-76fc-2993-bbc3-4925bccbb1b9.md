
# Series.Name Property (PowerPoint)

Returns or sets the name of the object. Read/write  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[Series](5c8c2d92-d8ca-4d21-e213-c374292275d4.md)** object.
