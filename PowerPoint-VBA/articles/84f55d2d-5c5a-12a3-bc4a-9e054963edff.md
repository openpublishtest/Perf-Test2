
# TextFrame2.Application Property (PowerPoint)

Returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the creator of the specified **TextFrame2** object. Read-only.


## Syntax

 _expression_. **Application**

 _expression_ An expression that returns a **TextFrame2** object.
