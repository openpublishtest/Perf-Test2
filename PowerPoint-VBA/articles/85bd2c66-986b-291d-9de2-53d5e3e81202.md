
# Point.Copy Method (PowerPoint)

If the point has a picture fill, copies the picture to the Clipboard.


## Syntax

 _expression_. **Copy**

 _expression_ A variable that represents a **[Point](e0137fdd-5632-88d7-a6c0-57a76717e736.md)** object.
