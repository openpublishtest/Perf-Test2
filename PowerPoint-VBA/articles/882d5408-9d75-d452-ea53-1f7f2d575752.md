
# LegendKey.MarkerBackgroundColorIndex Property (PowerPoint)

Returns or sets the marker background color as an index into the current color palette, or as one of the following  **[XlColorIndex](e9d26c5b-26f8-7baf-5f53-4beac789add3.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone**. Read/write **Long**.


## Syntax

 _expression_. **MarkerBackgroundColorIndex**

 _expression_ A variable that represents a **[LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.


## Remarks

This property applies only to line, scatter, and radar charts.
