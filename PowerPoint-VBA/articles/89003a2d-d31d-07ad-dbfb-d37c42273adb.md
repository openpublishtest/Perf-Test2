
# Trendline.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Trendline](74755c19-0a7d-cbbf-857e-78740adf6aa4.md)** object.
