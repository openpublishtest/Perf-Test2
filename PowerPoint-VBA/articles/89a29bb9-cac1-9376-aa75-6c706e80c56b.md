
# DataLabel.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[DataLabel](a17d23c5-0361-9129-28e5-b892f6966bda.md)** object.
