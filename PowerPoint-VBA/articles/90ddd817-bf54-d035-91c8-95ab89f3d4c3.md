
# HiLoLines.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[HiLoLines](77a7ae91-daf3-4c35-1f39-067d2698fb43.md)** object.
