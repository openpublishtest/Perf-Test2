
# ChartGroup.BinsUnderflowValue Property (PowerPoint)

If [ChartGroup.BinsUnderflowEnabled Property (PowerPoint)](42b53b36-5a40-ac5d-cf2c-7658128006ca.md) is **True**, specifies the value below which an underflow bin is displayed. Read/write  **Double**.


## Syntax

 _expression_. **BinsUnderflowValue**

 _expression_ A variable that represents a **ChartGroup** object.
