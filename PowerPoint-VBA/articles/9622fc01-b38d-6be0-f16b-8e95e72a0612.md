
# ChartFont.Name Property (PowerPoint)

Returns or sets the name of the object. Read/write  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[ChartFont](185dfaa0-4ed9-01d2-6584-b0838b50ef8c.md)** object.
