
# LegendEntry.Width Property (PowerPoint)

Returns the width, in points, of the object. Read-only  **Double**.


## Syntax

 _expression_. **Width**

 _expression_ A variable that represents a **[LegendEntry](c92ddccd-92a3-bec9-cdcd-efd82c77706b.md)** object.
