
# Gridlines.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[GridLines](10b45c4c-05a3-f722-15ca-ad0242625edb.md)** object.
