
# HiLoLines.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[HiLoLines](77a7ae91-daf3-4c35-1f39-067d2698fb43.md)** object.
