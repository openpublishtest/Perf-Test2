
# DataLabels.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[DataLabels](a0d0b0ec-6a12-9a5c-1026-1e1d85e488fa.md)** object.
