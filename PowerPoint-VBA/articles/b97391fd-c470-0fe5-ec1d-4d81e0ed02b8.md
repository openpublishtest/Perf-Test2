
# Coauthoring.MergeMode Property (PowerPoint)

Returns  **True** if the application is in merge mode. Read-only.


## Syntax

 _expression_. **MergeMode**

 _expression_ A variable that represents a **Coauthoring** object.


### Return Value

Boolean
