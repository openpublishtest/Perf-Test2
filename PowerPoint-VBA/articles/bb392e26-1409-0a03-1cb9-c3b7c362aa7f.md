
# Effect.Shape Property (PowerPoint)

Returns a  **[Shape](1da93849-99e0-827e-ced3-c6cf7f8569f3.md)** object that represents an animated shape.


## Syntax

 _expression_. **Shape**

 _expression_ A variable that represents an **Effect** object.


### Return Value

Shape
