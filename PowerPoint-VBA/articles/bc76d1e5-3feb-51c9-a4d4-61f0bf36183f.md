
# Shape.TextFrame2 Property (PowerPoint)

Returns the  **[TextFrame2](ae017598-8330-4673-db1a-53b284acb709.md)** object associated with the specified **[Shape](1da93849-99e0-827e-ced3-c6cf7f8569f3.md)** object that contains the alignment and anchoring properties for the specified shape. Read-only.


## Syntax

 _expression_. **TextFrame2**

 _expression_ An expression that returns a **Shape** object.


### Return Value

TextFrame2


## Remarks

Use the  **[TextRange](288c1209-d12d-fd7c-bc1a-6775d844ca6b.md)** property of the **TextFrame2** object to return the text in the text frame.

Use the  **[HasTextFrame](ea1a53e4-32d8-e51f-9e60-9ef719c0d973.md)** property to determine whether a shape contains a text frame before you attempt to get the **TextFrame2** property value.
