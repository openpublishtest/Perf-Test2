
# Chart.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.
