
# ShadowFormat.Blur Property (PowerPoint)

Returns or sets the blur radius of the specified shadow, in points. Read/write.


## Syntax

 _expression_. **Blur**

 _expression_ A variable that represents a **ShadowFormat** object.


### Return Value

Single
