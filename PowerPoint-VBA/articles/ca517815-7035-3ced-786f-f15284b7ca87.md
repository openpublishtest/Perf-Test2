
# DownBars Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](92238dae-e889-e765-4cf9-1d2253167b40.md)|When used without an object qualifier, returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the Microsoft PowerPoint application. |[Parent](6e4a65de-c892-c0cc-1ec8-c7929f12539a.md)|Returns the parent for the specified object. Read-only  **Object**.|
