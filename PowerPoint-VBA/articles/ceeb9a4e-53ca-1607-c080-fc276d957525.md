
# Comment Members (PowerPoint)
Represents a comment on a given slide or slide range. The  **Comment** object is a member of the **[Comments](1f29db7c-90fa-db9f-5229-136534ce803d.md)** collection object.

Represents a comment on a given slide or slide range. The  **Comment** object is a member of the **[Comments](1f29db7c-90fa-db9f-5229-136534ce803d.md)** collection object.

|**Name**|**Description**|
|:-----|:-----|
