
# ChartTitle.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[ChartTitle](21305a3b-1c77-d420-2156-79083189df03.md)** object.
