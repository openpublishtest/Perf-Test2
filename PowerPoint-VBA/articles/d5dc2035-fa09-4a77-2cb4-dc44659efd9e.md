
# Axis.Height Property (PowerPoint)

Returns the height, in points, of the object. Read-only  **Double**.


## Syntax

 _expression_. **Height**

 _expression_ A variable that represents an **[Axis](38d5e006-ac32-7bdb-f9f0-e8a858dcbf49.md)** object.
