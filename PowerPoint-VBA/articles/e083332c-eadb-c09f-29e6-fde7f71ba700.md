
# DisplayUnitLabel.ReadingOrder Property (PowerPoint)

Returns or sets an  **[XlReadingOrder](38062197-37e0-f116-f7fb-d6ebeba2ff12.md)** constant that represents the reading order for the specified object. Read/write **Long**.


## Syntax

 _expression_. **ReadingOrder**

 _expression_ A variable that represents a **[DisplayUnitLabel](4dd4df7d-91c1-9136-2d5b-cdb0794a7716.md)** object.
