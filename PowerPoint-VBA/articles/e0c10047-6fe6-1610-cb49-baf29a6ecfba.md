
# PlotArea.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[PlotArea](c1b991b8-8be2-5342-8b5c-814a2e99fec2.md)** object.
