
# Chart.HasTitle Property (PowerPoint)

 **True** if the axis or chart has a visible title. Read/write **Boolean**.


## Syntax

 _expression_. **HasTitle**

 _expression_ A variable that represents a **[Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.


## Remarks

A chart title is represented by a  **[ChartTitle](21305a3b-1c77-d420-2156-79083189df03.md)** object.
