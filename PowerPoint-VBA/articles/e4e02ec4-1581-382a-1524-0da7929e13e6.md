
# Trendlines.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **[Trendlines](8ac46695-aae0-3611-ebf7-c7339ea733ab.md)** object.
