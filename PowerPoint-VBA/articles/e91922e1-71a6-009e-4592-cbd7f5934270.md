
# ShapeRange.SmartArt Property (PowerPoint)

Returns the SmartArt diagram of the  **ShapeRange** object. Read-only.


## Syntax

 _expression_. **SmartArt**

 _expression_ A variable that represents a **ShapeRange** object.
