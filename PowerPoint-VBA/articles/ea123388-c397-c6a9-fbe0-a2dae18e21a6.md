
# DropLines.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[DropLines](b13b58c3-d00d-16d2-16ef-bcd3cae347c5.md)** object.
