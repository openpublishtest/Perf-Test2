
# ChartGroup.AxisGroup Property (PowerPoint)

Returns the type of axis group. Read/write  **[XlAxisGroup](775041e9-c965-a9b6-b5fb-cdebe4fb71c0.md)**.


## Syntax

 _expression_. **AxisGroup**

 _expression_ A variable that represents a **[ChartGroup](5caa5855-bd69-3fbc-f601-504e431a42e9.md)** object.
