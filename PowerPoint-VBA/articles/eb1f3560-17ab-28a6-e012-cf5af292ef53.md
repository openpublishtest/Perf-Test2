
# BulletFormat.Type Property (PowerPoint)

Represents the type of bullet. Read/write.


## Syntax

 _expression_. **Type**

 _expression_ A variable that represents a **BulletFormat** object.


### Return Value

PpBulletType


## Remarks

The value of the  **Type** property can be one of these **PpBulletType** constants.


||
|:-----|
|**ppBulletMixed**|
|**ppBulletNone**|
|**ppBulletNumbered**|
|**ppBulletPicture**|
|**ppBulletUnnumbered**|
