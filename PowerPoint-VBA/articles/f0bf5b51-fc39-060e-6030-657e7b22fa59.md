
# Axis.Type Property (PowerPoint)

Returns the axis type. Read-only  **[XlAxisType](6eb891d5-3b69-e0a4-90e5-0b21afb1eeaa.md)**.


## Syntax

 _expression_. **Type**

 _expression_ A variable that represents an **[Axis](38d5e006-ac32-7bdb-f9f0-e8a858dcbf49.md)** object.


## Remarks

When you use this property for the horizontal axis of scatter charts,  **xlCategory** will be returned as the axis type.
