
# ChartTitle.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartTitle](21305a3b-1c77-d420-2156-79083189df03.md)** object.
