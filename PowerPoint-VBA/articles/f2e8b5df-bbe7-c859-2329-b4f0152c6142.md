
# TickLabels.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[TickLabels](2ba878bf-3a76-1350-2bd4-615c2520f042.md)** object.
