
# ChartTitle.Width Property (PowerPoint)

Returns or sets a  **Double** value that represents the width, in points, of the object. Read-only.


## Syntax

 _expression_. **Width**

 _expression_ A variable that represents a **ChartTitle** object.


### Return Value

Double
