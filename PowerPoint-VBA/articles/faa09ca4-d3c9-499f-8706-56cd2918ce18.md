
# Trendlines Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](3ae91490-a646-1f9b-3b72-a24591f897d4.md)|When used without an object qualifier, returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Count](e4e02ec4-1581-382a-1524-0da7929e13e6.md)|Returns the number of objects in the collection. Read-only  **Long**.|
|[Creator](68677f70-e6ff-869d-218e-84cb0c98f4f2.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Parent](cfb3b38f-29da-79e4-1a73-47011a5bf6a9.md)|Returns the parent for the specified object. Read-only  **Object**.|
