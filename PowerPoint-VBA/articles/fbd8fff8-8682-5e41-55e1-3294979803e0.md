
# DataLabel.Text Property (PowerPoint)

Returns or sets the text for the specified object. Read/write  **String**.


## Syntax

 _expression_. **Text**

 _expression_ A variable that represents a **[DataLabel](a17d23c5-0361-9129-28e5-b892f6966bda.md)** object.
