
# PjWeekday Enumeration (Project)

Contains constants that specify the day of the week.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjFriday**|6|Friday.|
|**pjMonday**|2|Monday.|
|**pjSaturday**|7|Saturday.|
|**pjSunday**|1|Sunday.|
|**pjThursday**|5|Thursday.|
|**pjTuesday**|3|Tuesday.|
|**pjWednesday**|4|Wednesday.|
