
# PjResourceGraphStyle Enumeration (Project)

Contains constants that specify the resource graph style.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjArea**|1|Area.|
|**pjBar**|0|Bar.|
|**pjDoNotShow**|5|Do not show.|
|**pjLineGraph**|3|Line graph.|
|**pjStep**|2|Step.|
|**pjStepLine**|4|Step line.|
