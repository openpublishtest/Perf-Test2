
# Application.Quit Method (Project)

Exits Microsoft Project.


## Syntax

 _expression_. **Quit**( ** _SaveChanges_** )

 _expression_ A variable that represents an **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _SaveChanges_|Optional|**Long**|Specifies whether Project saves changes before quitting. Can be one of the following  **[PjSaveType](f4f7fa95-4be9-b564-bcd9-1c6c1cb3adeb.md)** constants: **pjDoNotSave**, **pjSave**, or **pjPromptSave**. The default is **pjPromptSave** for new project files and projects that have changed since the last save.|

## Example

The following example saves all open projects and then exits Project.


```
Sub SaveChangesAndQuit() 
 Quit SaveChanges:=pjSave 
End Sub
```
