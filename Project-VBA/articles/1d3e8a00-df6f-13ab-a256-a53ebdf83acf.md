
# Task.Baseline2FixedCostAccrual Property (Project)

Gets or sets when the  **Task** baseline2 accrues fixed costs. Read/write **Long**. Can be one of the **[PjAccrueAt](a86ac41f-9b7c-dd20-6d41-131b1c96af6b.md)** constants.


## Syntax

 _expression_. **Baseline2FixedCostAccrual**

 _expression_ A variable that represents a **Task** object.
