
# Shape.Table Property (Project)
Gets the  **ReportTable** object in the shape. Read-only[ReportTable](db9846c7-fd53-ae5a-7a43-35dfc60f4fe4.md).

## Syntax

 _expression_. **Table**

 _expression_ A variable that represents a **Shape** object.


## Property value

 **REPORTTABLE**
