
# ChildDrivers.Count Property (Project)

Gets the number of items in the  **[ChildDrivers](5260ab69-f81a-3466-c969-d84a2c60551e.md)** collection. Read-only **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **ChildDrivers** object.


## Remarks

If  **TotalDetectedCount** is greater than 5 then count is 0.

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](de8cee7b-a471-4c3f-2965-de6c3d47dda5.md)** property.
