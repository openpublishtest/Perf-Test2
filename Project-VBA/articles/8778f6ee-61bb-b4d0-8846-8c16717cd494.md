
# Application.CustomFieldDelete Method (Project)

Deletes the specified custom field.


## Syntax

 _expression_. **CustomFieldDelete**( ** _Field_** )

 _expression_ A variable that represents an **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Field_|Required|**Long**|A value representing a  **[PjField](f0df0929-921c-1f33-ab42-192efdaeb64d.md)** constant.|

### Return Value

 **Boolean**
