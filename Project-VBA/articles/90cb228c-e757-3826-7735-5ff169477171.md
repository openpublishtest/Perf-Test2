
# Subproject.Index Property (Project)

Gets the index of a  **Subproject** object in the **Subprojects** containing object. Read-only **Long**.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents a **Subproject** object.


## Remarks

The  **Index** properties of different objects are used in similar ways. For an example, see the **[Index](1213f55b-aca0-76ee-2e8a-2442a2c576e1.md)** property of the **Project** object.
