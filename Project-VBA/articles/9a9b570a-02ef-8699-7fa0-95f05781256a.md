
# PjShape Enumeration (Project)

Contains constants that specify the type of drawing.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjArc**|6|Arc.|
|**pjArrow**|3|Arrow.|
|**pjEllipse**|5|Ellipse.|
|**pjLine**|2|Line.|
|**pjOLEObject**|1|OLE object.|
|**pjPolygon**|7|Polygon.|
|**pjRectangle**|4|Rectangle.|
|**pjTextBox**|8|Text box.|
