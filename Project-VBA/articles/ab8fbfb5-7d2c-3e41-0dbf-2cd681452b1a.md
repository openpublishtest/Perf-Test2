
# Year.Count Property (Project)

Gets the number of days in the  **Year** object. Read-only **Integer**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **Year** object.


## Remarks

Use of the  **Count** property in most collection objects is similar. For an example, see the **[Assignments.Count](de8cee7b-a471-4c3f-2965-de6c3d47dda5.md)** property.
