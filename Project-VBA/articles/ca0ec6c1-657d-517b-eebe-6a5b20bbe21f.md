
# Shapes.Parent Property (Project)
Gets an object that represents the parent of the  **Shapes** object. Read-only **Object**.

## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **Shapes** object.


## Property value

 **OBJECT**
