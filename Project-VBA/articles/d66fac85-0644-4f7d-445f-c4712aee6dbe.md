
# WorkWeek.Finish Property (Project)

Gets or sets the finish date of a work week represented by the **WorkWeek** object. Read/write **Variant**.


## Syntax

 _expression_. **Finish**

 _expression_ An expression that returns a **WorkWeek** object.
