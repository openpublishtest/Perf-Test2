
# CodeMaskLevel Members (Project)
Represents a level in the code mask of an outline code definition. The  **CodeMaskLevel** object is a member of the **[CodeMask](4d0a22f4-fee9-8f4b-a0c0-7bc817ad3f6a.md)** collection.
