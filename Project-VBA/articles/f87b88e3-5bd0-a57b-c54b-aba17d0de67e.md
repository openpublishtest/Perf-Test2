
# Tasks.UniqueID Property (Project)

Gets a  **[Task](bc6bb4a5-95a6-9d1f-3e28-92b9548a544a.md)** object from its containing collection. Read-only **Task**.


## Syntax

 _expression_. **UniqueID**( ** _Index_** )

 _expression_ A variable that represents a **Tasks** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required|**Long**|The position of the object within its containing collection.|
