
# Page.IsLeading Property (Publisher)

 **True** if the specified **Page** object is a leading page of a two-page spread. Read-only **Boolean**.


## Syntax

 _expression_. **IsLeading**

 _expression_A variable that represents an  **Page** object.


### Return Value

Boolean
