
# LayoutGuides Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](dd90b873-f106-6eb4-9c17-d41cf698c24b.md)|Used without an object qualifier, this property returns an  **[Application](acfc7efb-e6a5-a89a-3aee-3cb4af2f3508.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [ColumnGutterWidth](1c8fd297-1164-da50-cee8-390263cce5b0.md)|Returns or sets the width of the column gutters that are used by the  **LayoutGuides** object to aid in the process of laying out design elements. Read/write **Single**.|
| [Columns](926f7d06-0ff7-bf4a-033e-6b310ef9f400.md)|Returns or sets a  **Long** that represents the number of guide columns on a page or the number of columns in a text frame. Read/write.|
| [GutterCenterlines](7a5b1aef-85c7-548f-15e9-2c3b7327b439.md)|Returns or sets a value that specifies whether to add a center line between the columns and rows of the gutter guides in a master page. Read/write  **Boolean**.|
| [HorizontalBaseLineOffset](b80d2114-8132-db13-a50d-ce904dbe5919.md)|Returns a  **Single** that represents the horizontal baseline offset of the specified **LayoutGuides** object. Read/Write.|
| [HorizontalBaseLineSpacing](19899a25-c1a5-9c81-f022-d842a3d6c7d8.md)|Returns a  **Single** that represents the horizontal baseline spacing of the specified **LayoutGuides** object. Read/write.|
| [MarginBottom](9d11c4d9-8f53-7882-be40-200833a29fb6.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the bottom edge of a cell, text frame, or page. Read/write.|
| [MarginLeft](02d1a544-3e41-3875-3027-61bdc465e89b.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the left edge of a cell, text frame, or page. Read/write.|
| [MarginRight](5dbfc999-59d6-c9d0-4d9d-bc1a4ee622aa.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the right edge of a cell, text frame, or page. Read/write.|
| [MarginTop](f0b4f600-6c79-060b-edd5-82f07f78770a.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the top edge of a cell, text frame, or page. Read/write.|
| [MirrorGuides](8e6ff709-21e0-2286-5d75-c7ebea05fd26.md)|Returns or sets a  **Boolean** indicating whether Microsoft Publisher creates mirror guide positions for a book fold publication. **True** if Publisher creates mirror guide positions for separate left and right pages in a book fold publication; **False** if the same margin, row, and column guides are applied to all pages in the publication. Read/write.|
| [Parent](ed45fad9-e402-d799-b5de-58756653f393.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](95e88f5a-b3dc-272e-7c1d-5282c97ae11e.md)** object, returns a **[Shape](666cb7f0-62a8-f419-9838-007ef29506ee.md)** object representing the parent shape of the text frame. Read-only.|
| [RowGutterWidth](a7629683-68d2-4953-4c95-7e79e431f9c4.md)|Returns or sets the width of the row gutters that are used by the  **LayoutGuides** object to aid in the process of laying out design elements. Read/write **Single**.|
| [Rows](a42286ef-d955-c39d-49a4-b0e54b4d1cec.md)|Sets or returns a  **Long** that represents the number of rows in a layout guide. Read/write.|
| [VerticalBaseLineOffset](9a2f031c-4469-ca26-3e79-dfa556762e05.md)|Returns a  **Single** that represents the vertical baseline offset of the specified **LayoutGuides** object. Read/write.|
| [VerticalBaseLineSpacing](49391fbd-86c0-b53f-ff57-009af9341e74.md)|Returns a  **Single** that represents the vertical baseline spacing of the specified **LayoutGuides** object. Read/write.|
