
# Section Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](4d8dc397-fd29-9fca-4252-5f93c6ff6328.md)|Used without an object qualifier, this property returns an  **[Application](acfc7efb-e6a5-a89a-3aee-3cb4af2f3508.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [ContinueNumbersFromPreviousSection](a3d64f14-dc65-4fb1-5079-0fdf2e3f8f38.md)| **True** if the specified section continues the numbering from the prvious section. Read/write **Boolean**.|
| [PageNumberFormat](5b64a352-2fd8-9e19-3425-a7984dd67edd.md)|Sets or returns a  **PbPageNumberFormat** constant that reperesents the formatting of the page numbering. Read/write.|
| [PageNumberStart](f4124b7d-4a90-2489-13da-947df0c34a3d.md)|Sets or returns the page number that the specified section starts with. Read/write  **Long**.|
| [Parent](081f4653-2aff-22a3-f973-e86dade13de2.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](95e88f5a-b3dc-272e-7c1d-5282c97ae11e.md)** object, returns a **[Shape](666cb7f0-62a8-f419-9838-007ef29506ee.md)** object representing the parent shape of the text frame. Read-only.|
| [ShowHeaderFooterOnFirstPage](6c814884-9bee-72ae-3a40-5118bebd6f02.md)| **True** if the header and footer of the specified section will be visible. Read/write **Boolean**.|
| [StartPageIndex](531d6724-3f18-bd2a-974c-5b58d64c2af1.md)|Returns the page number of the page that the specified  **Section** object begins on. Read-only.|
|Name|Description|
