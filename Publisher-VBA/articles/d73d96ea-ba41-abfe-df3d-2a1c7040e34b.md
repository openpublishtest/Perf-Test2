
# Sections Members (Publisher)
A collection of all the  **Section** objects in the document.

## Methods



|**Name**|**Description**|
|:-----|:-----|
| [Add](d5343281-375a-4817-8353-a3fa8925a2ce.md)|Adds a new  **Section** object to the specified **Sections** collection and returns the new **Section** object.|
|Name|Description|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](472f91a6-d228-4bb5-106b-4677b883ece1.md)|Used without an object qualifier, this property returns an  **[Application](acfc7efb-e6a5-a89a-3aee-3cb4af2f3508.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Count](39a8848b-e528-7635-8f02-57f200f6a4c9.md)|Returns a  **Long** that represents the number of items in the specified collection.|
| [Item](0ca9577b-f046-fc3d-1f4e-412be7114d5c.md)|Returns an individual object from a specified collection. Read-only.|
| [Parent](29df8618-fc04-ba78-19da-c954c6111659.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](95e88f5a-b3dc-272e-7c1d-5282c97ae11e.md)** object, returns a **[Shape](666cb7f0-62a8-f419-9838-007ef29506ee.md)** object representing the parent shape of the text frame. Read-only.|
