
# PbRecipientListFileType Enumeration (Publisher)

Constants passed to the  ** [MailMerge.ExportRecipientList](230d0f66-7368-51b7-8233-3fd54cfd0fe4.md)** method that specify the type of file in which to save an exported mail-merge recipient list.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **pbAsCsvFile**|1|Save as comma-delimited CSV file.|
| **pbAsMdbFile**|0|Save as Microsoft Office Access MDB file.|
