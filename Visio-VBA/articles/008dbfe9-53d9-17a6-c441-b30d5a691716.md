
# ContainerProperties.SetMargin Method (Visio)

Sets the gap between the container and member shapes to the specified size, in the specified units.


## Syntax

 _expression_ . **SetMargin**( **_MarginUnits_** , **_MarginSize_** )

 _expression_ A variable that represents a **[ContainerProperties](b94f758f-58f7-f1ef-c03b-761e26c11017.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _MarginUnits_|Required| **[VisUnitCodes](fce91c1b-d5c2-6522-2446-0b8f6cacbc84.md)**|The units in which to measure the margin.|
| _MarginSize_|Required| **Double**|The size of the margin.|

### Return Value

 **Nothing**
