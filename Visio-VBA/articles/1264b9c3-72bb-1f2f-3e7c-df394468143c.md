
# MovedSelectionEvent.ReplacementShape Property (Visio)

Returns a  **[Shape](da7a8872-4ebb-a607-e0ed-eebf68ff5630.md)** object that represents the shape that is in the source page and that replaces the source-page shapes or selection being moved to a sub-process page. Read-only.


## Syntax

 _expression_ . **ReplacementShape**

 _expression_ A variable that represents a **[MovedSelectionEvent](e29b0f23-08b6-7c63-9955-4a989c6e6892.md)** object.


### Return Value

 **Shape**
