
# ValidationRuleSets.ObjectType Property (Visio)

Returns  **visObjTypeValidationRuleSets** , the type of this object. Read-only.


## Syntax

 _expression_ . **ObjectType**

 _expression_ A variable that represents a **[ValidationRuleSets](f08d7f04-13ec-8175-2aa6-94b0b67ee76b.md)** object.


### Return Value

 **visObjTypeValidationRuleSets**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](47d8b8f5-e402-7305-69d7-79d55d5c7f9e.md)** enumeration.
