
# ValidationRuleSet.Application Property (Visio)

Returns the instance of Microsoft Visio that is associated with an object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[ValidationRuleSet](cd2fc58a-5d7c-cf31-7aab-41bdeee9f105.md)** object.


### Return Value

 **[Application](5b3c8939-793f-116f-11b8-1d4170d95a63.md)**
