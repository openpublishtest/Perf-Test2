
# Hyperlink Methods (Visio)

|**Name**|**Description**|
|:-----|:-----|

|[Follow](e415caa8-68b9-5c96-71f0-599655dc6cf3.md)|Causes Microsoft Visio to navigate to a hyperlink.|
