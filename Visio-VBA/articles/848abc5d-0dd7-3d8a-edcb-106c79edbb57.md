
# ValidationRuleSets.Document Property (Visio)

Returns the  **[Document](21640062-13a2-a2b2-7c61-7e707671207c.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_ . **Document**

 _expression_ A variable that represents a **[ValidationRuleSets](f08d7f04-13ec-8175-2aa6-94b0b67ee76b.md)** object.


### Return Value

 **Document**
