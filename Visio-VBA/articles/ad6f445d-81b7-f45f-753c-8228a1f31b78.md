
# Addon.NameU Property (Visio)

Specifies the universal name of an  **Addon** object. Read-only.


## Syntax

 _expression_ . **NameU**

 _expression_ A variable that represents an **Addon** object.


### Return Value

String


## Remarks

You can get, but not set, the  **NameU** property of an **Addon** object.
