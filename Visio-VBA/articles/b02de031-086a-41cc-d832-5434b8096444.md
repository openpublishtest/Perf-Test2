
# Application.AfterReplaceShapes Event (Visio)

Occurs after a shape-replacement operation.


## Syntax

 _expression_ . **AfterReplaceShapes**_(sel)_

 _expression_ A variable that represents an **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|||||
| _sel_|Required|IVSELECTION|A [Selection](e5734140-6dbe-7de8-9695-1a22fb4ac628.md) object that represents the replacement shape or shapes.|
