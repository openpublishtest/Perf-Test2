
# ValidationIssues.ItemFromID Property (Visio)

Returns the  **[ValidationIssue](b1e93738-48da-cf68-24ad-dd03f79ad152.md)** object that matches the specified ID. Read-only.


## Syntax

 _expression_ . **ItemFromID**( **_IssueID_** )

 _expression_ A variable that represents a **[ValidationIssues](13362aa2-7e09-14ed-8aa9-bf2a93edf302.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _IssueID_|Required| **Long**|The ID of the object to retrieve.|

### Return Value

 **ValidationIssue**
