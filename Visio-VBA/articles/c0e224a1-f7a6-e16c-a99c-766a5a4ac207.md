
# ContainerProperties.GetMargin Method (Visio)

Returns the minimal distance, in the specified units, between the edges of the container or list and those of its member shapes.


## Syntax

 _expression_ . **GetMargin**( **_MarginUnits_** )

 _expression_ A variable that represents a **[ContainerProperties](b94f758f-58f7-f1ef-c03b-761e26c11017.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _MarginUnits_|Required| **[VisUnitCodes](fce91c1b-d5c2-6522-2446-0b8f6cacbc84.md)**|The units in which to measure the margin.|

### Return Value

 **Double**
