
# ValidationRules.Document Property (Visio)

Returns the  **[Document](21640062-13a2-a2b2-7c61-7e707671207c.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_ . **Document**

 _expression_ A variable that represents a **[ValidationRules](e7a1a5c6-02a7-2dc2-7a73-cc84821e077e.md)** object.


### Return Value

 **Document**
