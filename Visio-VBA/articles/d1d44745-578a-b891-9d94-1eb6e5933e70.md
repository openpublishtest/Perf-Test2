
# ValidationRule.Document Property (Visio)

Returns the  **[Document](21640062-13a2-a2b2-7c61-7e707671207c.md)** object that is associated with an object. Read-only.


## Syntax

 _expression_ . **Document**

 _expression_ A variable that represents a **[ValidationRule](c9efb9b4-10b0-b6aa-cc78-2a01fd3e8357.md)** object.


### Return Value

 **Document**
