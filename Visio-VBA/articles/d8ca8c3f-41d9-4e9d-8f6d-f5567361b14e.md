
# VisSVGExportFormat Enumeration (Visio)

Constants passed to and returned by the [ApplicationSettings.SVGExportFormat](9e7ca1cb-5ace-b75b-0e59-61566b9a0169.md) property that specify whether to include both SVG elements and Visio elements when saving Microsoft Visio documents to the SVG file format.


## Members



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
||||
| **visSVGExcludeVisioElements**| **1**|Include SVG elements only.|
| **visSVGIncludeVisioElements**| **0**|Include both SVG elements and Visio elements. This is the default.|
