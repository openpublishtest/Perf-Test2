
# ValidationRules.ObjectType Property (Visio)

Returns  **visObjTypeValidationRules** , the type of this object. Read-only.


## Syntax

 _expression_ . **ObjectType**

 _expression_ A variable that represents a **[ValidationRules](e7a1a5c6-02a7-2dc2-7a73-cc84821e077e.md)** object.


### Return Value

 **visObjTypeValidationRules**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](47d8b8f5-e402-7305-69d7-79d55d5c7f9e.md)** enumeration.
