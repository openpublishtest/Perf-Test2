
# ServerPublishOptions.Application Property (Visio)

Returns the instance of Microsoft Visio that is associated with an object. Read-only.


 **Note**  This Visio object or member is available only to licensed users of Visio Professional 2013.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[ServerPublishOptions](69e71212-4ca3-9fa6-6af3-8f07af540140.md)** object.


### Return Value

 **[Application](5b3c8939-793f-116f-11b8-1d4170d95a63.md)**
