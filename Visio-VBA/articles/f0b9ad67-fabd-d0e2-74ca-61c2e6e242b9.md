
# VisPublishDataRecordsets Enumeration (Visio)

Specifies constants that define which data recordsets to publish; passed to the  **[ServerPublishOptions.SetRecordsetsToPublish](c79a8677-e4f0-9eff-9eda-72b11d0af240.md)** method and returned by the **[ServerPublishOptions.GetRecordsetsToPublish](d0f1981d-f0ef-12dc-a0aa-562ef38a7aec.md)** method.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **visPublishDataRecordsetAll**|0|Publish all data recordsets in the document.|
| **visPublishDataRecordsetNone**|1|Publish none of the data recordsets in the document.|
| **visPublishDataRecordsetSelect**|2|Publish selected data recordsets.|
