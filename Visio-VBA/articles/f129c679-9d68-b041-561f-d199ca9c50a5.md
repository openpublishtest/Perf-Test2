
# ServerPublishOptions.ObjectType Property (Visio)

Returns  **visObjTypeServerPublishOptions** , the type of this object. Read-only.


 **Note**  This Visio object or member is available only to licensed users of Visio Professional 2013.


## Syntax

 _expression_ . **ObjectType**

 _expression_ A variable that represents a **[ServerPublishOptions](69e71212-4ca3-9fa6-6af3-8f07af540140.md)** object.


### Return Value

 **visObjTypeServerPublishOptions**


## Remarks

Constants that represent object types are prefixed with  **visObjType** and are declared by the Visio type library in the **[VisObjectTypes](47d8b8f5-e402-7305-69d7-79d55d5c7f9e.md)** enumeration.
