
# ValidationRuleSets.Application Property (Visio)

Returns the instance of Microsoft Visio that is associated with an object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[ValidationRuleSets](f08d7f04-13ec-8175-2aa6-94b0b67ee76b.md)** object.


### Return Value

 **[Application](5b3c8939-793f-116f-11b8-1d4170d95a63.md)**
