
# MovedSelectionEvent.Application Property (Visio)

Returns the instance of Microsoft Visio that is associated with an object. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ A variable that represents a **[MovedSelectionEvent](e29b0f23-08b6-7c63-9955-4a989c6e6892.md)** object.


### Return Value

 **[Application](5b3c8939-793f-116f-11b8-1d4170d95a63.md)**
