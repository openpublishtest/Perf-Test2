
# DropCap.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **DropCap** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[DropCap](79daea90-657b-43db-34e3-08f7aed74591.md)** object.
