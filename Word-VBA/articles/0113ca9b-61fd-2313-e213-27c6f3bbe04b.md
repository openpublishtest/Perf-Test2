
# Walls.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Walls](e98c7218-b944-12bb-caf9-daecee4b6c0c.md)** object.
