
# ShapeNodes.Count Property (Word)

Returns a  **Long** that represents the number of shape nodes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[ShapeNodes](f2e13db2-102f-1a14-fd7a-d179f63e513e.md)** collection.
