
# StyleSheets.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **StyleSheets** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[StyleSheets](4b3da354-38a6-5758-3080-82a940b668c9.md)** collection.
