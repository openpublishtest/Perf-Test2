
# Options.MoveFromTextColor Property (Word)

Returns or sets a  **[WdColorIndex](4c4a9a72-79d6-79ab-9e82-377abe2fb300.md)** constant that represents the color of moved text. Read/write.


## Syntax

 _expression_ . **MoveFromTextColor**

 _expression_ An expression that returns an **[Options](873b7b99-3fe1-fd89-9ece-a9355cb827dc.md)** object.
