
# Sources.Count Property (Word)

Returns the number of items in the  **Sources** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Sources](92a74894-1c69-0ff1-749d-baa49b45a174.md)** object.
