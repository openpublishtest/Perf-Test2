
# Application.Height Property (Word)

Returns or sets the height of the active document window in pixels. Read/write  **Long** .


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents an **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.
