
# Axis.Height Property (Word)

Returns the height, in points, of the object. Read-only  **Double** .


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents an **[Axis](3a7ad7d8-d397-a79a-eb6a-a5f0822cbe5d.md)** object.
