
# ShapeRange.ID Property (Word)

Returns the identification type for the range of shapes. Read-only  **Long** .


## Syntax

 _expression_ . **ID**

 _expression_ Required. A variable that represents a **[ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
