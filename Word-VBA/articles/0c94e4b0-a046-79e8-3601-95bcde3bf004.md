
# Subdocuments.Count Property (Word)

Returns a  **Long** that represents the number of subdocuments in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Subdocuments](8e14fced-19b3-2794-39a3-9e5ec15dd0ad.md)** collection.
