
# Shape.ID Property (Word)

Returns the identification type for the specified shape. Read-only  **Long** .


## Syntax

 _expression_ . **ID**

 _expression_ Required. A variable that represents a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.
