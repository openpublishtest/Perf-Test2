
# Range.Characters Property (Word)

Returns a  **[Characters](6d22ae7a-128d-134d-9136-1cdd5a8d9941.md)** collection that represents the characters in a range. Read-only.


## Syntax

 _expression_ . **Characters**

 _expression_ A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](http://msdn.microsoft.com/library/28f76384-f495-9640-a7c8-10ada3fac727%28Office.15%29.aspx).
