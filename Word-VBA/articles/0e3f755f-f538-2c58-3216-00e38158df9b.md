
# TablesOfContents.Item Method (Word)

Returns an individual  **TableOfContents** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[TablesOfContents](d0d0e5fc-e443-31ae-e1a9-15b945f1e318.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

TableOfContents
