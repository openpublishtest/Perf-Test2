
# Research.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Research** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Research](454e1fd6-0e52-84df-7d15-04fda00b177b.md)** object.
