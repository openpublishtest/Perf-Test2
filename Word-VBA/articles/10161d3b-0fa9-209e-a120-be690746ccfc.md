
# Selection.Range Property (Word)

Returns a  **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object that represents the portion of a document that's contained in the specified object.


## Syntax

 _expression_ . **Range**

 _expression_ Required. A variable that represents a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.
