
# Window.Width Property (Word)

Returns or sets the width of the specified document window, in points. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Window](d92f83f9-ae44-56c0-4584-7a9359253c6d.md)** object.
