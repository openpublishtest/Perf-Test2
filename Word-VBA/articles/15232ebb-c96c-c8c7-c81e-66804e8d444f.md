
# ShadowFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ShadowFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ShadowFormat](2a179f0b-ec18-c3dd-dd73-51b18f42e0e2.md)** object.
