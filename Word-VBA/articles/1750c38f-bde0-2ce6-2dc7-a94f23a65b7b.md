
# OMathFunction.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathFunction** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathFunction](2347ec7b-5e1a-8039-5fd0-195c08860cb5.md)** object.
