
# CalloutFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CalloutFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[CalloutFormat](d54764e6-d761-582b-aa0a-baebd3a7cf6a.md)** object.
