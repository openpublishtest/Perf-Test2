
# Section.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Section** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Section](3fe563d8-fc05-c17a-e67b-c50eea7e7f13.md)** object.
