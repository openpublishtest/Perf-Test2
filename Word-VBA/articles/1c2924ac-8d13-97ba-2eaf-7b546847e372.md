
# MailingLabel.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MailingLabel** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[MailingLabel](9dd073b7-5d53-0f1e-f19a-9abf6427b3f2.md)** object.
