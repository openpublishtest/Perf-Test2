
# Selection.Shading Property (Word)

Returns a  **[Shading](e136509a-1be1-29e4-7b37-1faf659e37ba.md)** object that refers to the shading formatting for the specified selection.


## Syntax

 _expression_ . **Shading**

 _expression_ Required. A variable that represents a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.
