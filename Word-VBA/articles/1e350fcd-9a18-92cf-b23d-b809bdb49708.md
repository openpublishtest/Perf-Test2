
# InlineShape.Height Property (Word)

Returns or sets the height of an inline shape. Read/write  **Single** .


## Syntax

 _expression_ . **Height**

 _expression_ An expression that represents a **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
