
# Shading.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Shading** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Shading](e136509a-1be1-29e4-7b37-1faf659e37ba.md)** object.
