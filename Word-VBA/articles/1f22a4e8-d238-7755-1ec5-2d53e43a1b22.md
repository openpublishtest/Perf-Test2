
# TextColumn.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TextColumn** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TextColumn](660614a8-ad5b-dae4-887e-0f75e1172c10.md)** object.
