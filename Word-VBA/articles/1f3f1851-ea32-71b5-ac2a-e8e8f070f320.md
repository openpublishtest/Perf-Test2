
# OMathLimUpp.ToLimLow Method (Word)

Converts an equation from the upper limit to the lower limit.


## Syntax

 _expression_ . **ToLimLow**

 _expression_ An expression that returns a **OMathLimUpp** object.


### Return Value

OMathFunction
