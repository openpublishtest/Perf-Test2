
# ChartTitle.Shadow Property (Word)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.
