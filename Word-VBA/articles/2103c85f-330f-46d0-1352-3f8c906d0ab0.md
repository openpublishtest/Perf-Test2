
# Hyperlink.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Hyperlink** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Hyperlink](af785a9e-081a-e359-705f-04f490304e2e.md)** object.
