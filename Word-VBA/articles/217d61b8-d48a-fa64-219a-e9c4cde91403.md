
# LegendEntries.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[LegendEntries](3d130934-8a2d-a2f5-b609-3ab34f406dc4.md)** object.
