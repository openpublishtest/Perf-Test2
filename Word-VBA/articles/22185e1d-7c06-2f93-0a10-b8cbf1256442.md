
# TextColumns.Count Property (Word)

Returns a  **Long** that represents the number of text columns in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[TextColumns](00b62c93-db7d-00b9-cc84-9a21e427d0cd.md)** collection.
