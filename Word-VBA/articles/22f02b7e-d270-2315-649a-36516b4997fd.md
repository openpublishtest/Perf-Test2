
# AutoCorrectEntry.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **AutoCorrectEntry** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[AutoCorrectEntry](33173958-42eb-00ef-7f37-41f95ed47f87.md)** object.
