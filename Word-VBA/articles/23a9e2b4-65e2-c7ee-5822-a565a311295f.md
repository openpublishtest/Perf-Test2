
# Legend.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Legend](f0122074-87b7-0225-3c6c-406103fa4c29.md)** object.
