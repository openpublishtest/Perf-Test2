
# RecentFile.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[RecentFile](c8d7a06d-c340-2d35-d4a9-5d0cd4a07aab.md)** object.
