
# TabStops.Count Property (Word)

Returns a  **Long** that represents the number of tab stops in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[TabStops](2d3bcac4-db8c-05fe-1cc1-5d90774f84fb.md)** collection.
