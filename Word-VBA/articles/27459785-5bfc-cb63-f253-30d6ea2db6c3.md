
# TablesOfFigures.Format Property (Word)

Returns or sets the formatting for the tables of figures in the specified document. Read/write  **WdTofFormat** .


## Syntax

 _expression_ . **Format**

 _expression_ Required. A variable that represents a **[TablesOfFigures](2a5b3c3d-bb25-e31d-e7d3-b011732de6fb.md)** collection.
