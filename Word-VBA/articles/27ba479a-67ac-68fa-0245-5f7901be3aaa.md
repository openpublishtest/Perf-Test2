
# CheckBox.Value Property (Word)

 **True** if the check box is selected. Read/write **Boolean** .


## Syntax

 _expression_ . **Value**

 _expression_ Required. A variable that represents a **[CheckBox](e72b57b7-0328-9e78-94ca-ab7fb3c64afb.md)** object.
