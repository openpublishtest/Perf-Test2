
# CustomLabels.Count Property (Word)

Returns a  **Long** that represents the number of items in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[CustomLabels](407e75b5-4116-fdc7-f0c1-dfd3809cdb41.md)** collection.
