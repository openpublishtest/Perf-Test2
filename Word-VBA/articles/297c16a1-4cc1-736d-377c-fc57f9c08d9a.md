
# ChartFont.Name Property (Word)

Returns or sets the name of the object. Read/write  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **[ChartFont](2ca7fb97-fa22-dec1-6978-8ebb6d8aad7c.md)** object.
