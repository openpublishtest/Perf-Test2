
# Bookmarks.Count Property (Word)

Returns the number of items in the  **Bookmarks** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Bookmarks](827bed64-3034-0eb4-401d-f117cdb98898.md)** object.
