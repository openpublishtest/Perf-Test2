
# TableStyle.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TableStyle** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TableStyle](4f1f4489-0ef7-dff0-8f2a-77f87937f3ad.md)** object.
