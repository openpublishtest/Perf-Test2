
# Columns.Borders Property (Word)

Returns a  **[Borders](6dd1d4cc-2dcf-22c7-a299-4721a5543ba3.md)** collection that represents all the borders for the specified columns.


## Syntax

 _expression_ . **Borders**

 _expression_ Required. A variable that represents a **[Columns](7c2d1353-cbc4-a162-83a1-6cac1300266f.md)** collection.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](http://msdn.microsoft.com/library/28f76384-f495-9640-a7c8-10ada3fac727%28Office.15%29.aspx).
