
# InlineShapes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **InlineShapes** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[InlineShapes](88c632b2-80de-c96a-8879-a98461b38bd0.md)** collection.
