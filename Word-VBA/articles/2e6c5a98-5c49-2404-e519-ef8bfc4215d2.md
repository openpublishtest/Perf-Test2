
# Bibliography Object (Word)

Represents the list of sources available attached to the document (in the current list) or the list of sources available in the application (in the master list).


## Remarks

Biblographies are composed of sources. Also see the  **[Source](f90108a8-6432-a700-86ce-7b8f9e9c034b.md)** object and the **[Sources](92a74894-1c69-0ff1-749d-baa49b45a174.md)** collection. For more information, see[Working with Bibliographies](http://msdn.microsoft.com/library/ce05a0bd-bacd-16e1-0ab0-793a47a15da5%28Office.15%29.aspx).
