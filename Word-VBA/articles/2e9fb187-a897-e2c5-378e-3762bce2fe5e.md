
# TableOfAuthorities.Delete Method (Word)

Deletes the specified table of authorities.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[TableOfAuthorities](abd7d600-8b20-0752-4629-8a4f5193dd5d.md)** collection.
