
# ThreeDFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ThreeDFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ThreeDFormat](d397e780-a53d-0cc3-7a02-b40397253e91.md)** object.
