
# InlineShape.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **InlineShape** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
