
# DataLabels.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[DataLabels](a7676f18-b1f2-1e11-9489-863cb85c1669.md)** object.
