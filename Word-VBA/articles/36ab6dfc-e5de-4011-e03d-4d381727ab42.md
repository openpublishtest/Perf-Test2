
# Font.Line Property (Word)

Returns a [LineFormat](28fabccb-d03f-3466-9d07-ea3ebc4cdd11.md) object that specifies the formatting for a line. Read/write.


## Syntax

 _expression_ . **Line**

 _expression_ An expression that returns a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
