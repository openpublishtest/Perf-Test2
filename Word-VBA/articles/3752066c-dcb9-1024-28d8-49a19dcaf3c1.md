
# StoryRanges.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **StoryRanges** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[StoryRanges](131b04b0-e4a8-8969-0a4b-e5b3793af03d.md)** collection.
