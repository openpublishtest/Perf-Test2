
# Interior.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[Interior](6fc3e311-a7c9-bfa9-7459-9cea177b08e5.md)** object.
