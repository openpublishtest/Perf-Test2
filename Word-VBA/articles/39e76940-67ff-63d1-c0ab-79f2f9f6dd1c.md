
# Rectangle.Height Property (Word)

Returns a  **Long** that represents the height of a rectangle, in pixels.


## Syntax

 _expression_ . **Top**

 _expression_ An expression that represents a **[Rectangle](90ad4f48-2051-38f9-9b2e-a14bd38478be.md)** object.
