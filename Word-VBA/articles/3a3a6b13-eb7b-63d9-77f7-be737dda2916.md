
# ShapeNodes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ShapeNodes** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ShapeNodes](f2e13db2-102f-1a14-fd7a-d179f63e513e.md)** collection.
