
# AddIn.Name Property (Word)

Returns the name of an add-in. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ An expression that returns an **[AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** object.
