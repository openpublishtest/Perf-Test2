
# TwoInitialCapsException.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TwoInitialCapsException** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TwoInitialCapsException](48e89297-4137-960b-a92a-2a70929e298a.md)** object.
