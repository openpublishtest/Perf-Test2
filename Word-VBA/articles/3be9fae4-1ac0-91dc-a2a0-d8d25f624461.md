
# OMathFunc.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathFunc** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathFunc](c9acb683-6616-5f2e-d459-ea67b843c5d8.md)** object.
