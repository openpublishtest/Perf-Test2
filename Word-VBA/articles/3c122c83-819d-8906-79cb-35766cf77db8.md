
# FileConverter.Name Property (Word)

Returns name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[FileConverter](41af2a9b-75cc-253d-4954-4fb42c88530f.md)** object.
