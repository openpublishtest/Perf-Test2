
# Cells.Count Property (Word)

Returns the number of items in the  **Cells** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Cells](ceaa5b45-518d-d6ea-1ce8-5a34f6e37046.md)** collection.
