
# EmailSignatureEntries.Count Property (Word)

Returns a  **Long** that represents the number of e-mail signature entries in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents an **[EmailSignatureEntries](42a63f45-f989-be32-e75a-059c9a77c6f1.md)** collection.
