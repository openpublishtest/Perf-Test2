
# HeadingStyles.Item Method (Word)

Returns an individual  **HeadingStyle** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[HeadingStyles](be882a12-1264-8f7e-415b-b8bcbf28e703.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

HeadingStyle
