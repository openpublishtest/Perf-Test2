
# Tables.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Tables** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Tables](068a3d0f-0b19-3927-cb0a-7fb0d0fd8e52.md)** collection.
