
# ChartGroups.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[ChartGroups](37136fbd-8740-c817-9666-993bc5d4c847.md)** object.
