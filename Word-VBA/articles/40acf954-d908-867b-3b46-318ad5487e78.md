
# Hyperlink.Range Property (Word)

Returns a  **Range** object that represents the portion of a document that is contained within a hyperlink.


## Syntax

 _expression_ . **Range**

 _expression_ Required. A variable that represents a **[Hyperlink](af785a9e-081a-e359-705f-04f490304e2e.md)** object.
