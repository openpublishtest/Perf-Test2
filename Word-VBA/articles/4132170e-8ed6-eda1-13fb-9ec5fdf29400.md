
# Floor.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Floor](01d277eb-501b-09e5-65b8-83506c76ac05.md)** object.
