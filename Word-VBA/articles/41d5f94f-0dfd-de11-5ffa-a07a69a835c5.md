
# TablesOfFigures.Item Method (Word)

Returns an individual  **TableOfFigures** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[TablesOfFigures](2a5b3c3d-bb25-e31d-e7d3-b011732de6fb.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

TableOfFigures
