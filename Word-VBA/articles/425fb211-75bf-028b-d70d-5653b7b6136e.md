
# Column.Cells Property (Word)

Returns a  **[Cells](ceaa5b45-518d-d6ea-1ce8-5a34f6e37046.md)** collection that represents the table cells in a table column. Read-only.


## Syntax

 _expression_ . **Cells**

 _expression_ A variable that represents a **[Column](49d68571-2a57-6795-34b9-eb09aeb43043.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](http://msdn.microsoft.com/library/28f76384-f495-9640-a7c8-10ada3fac727%28Office.15%29.aspx).
