
# FileConverters.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FileConverters** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[FileConverters](b9b8fc53-1c8e-224d-726a-4edf172ca647.md)** collection.
