
# AddIn.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _expression_ . **Index**

 _expression_ Required. A variable that represents an **[AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** object.
