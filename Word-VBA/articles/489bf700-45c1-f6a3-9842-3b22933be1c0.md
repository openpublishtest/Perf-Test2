
# Endnotes.Count Property (Word)

Returns a  **Long** that represents the number of endnotes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents an **[Endnotes](32676579-dd41-e83d-a305-fcc2b7cb4f64.md)** collection.
