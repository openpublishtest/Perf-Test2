
# HeadersFooters.Count Property (Word)

Returns a  **Long** that represents the number of headers and/or footers in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[HeadersFooters](41dbbaa7-f139-3d3c-54d4-03a57ab8417a.md)** collection.
