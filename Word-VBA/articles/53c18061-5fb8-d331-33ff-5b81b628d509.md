
# Range.Find Property (Word)

Returns a  **[Find](da822788-cad5-992a-a835-18cc574cc324.md)** object that contains the criteria for a find operation. Read-only.


## Syntax

 _expression_ . **Find**

 _expression_ A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.
