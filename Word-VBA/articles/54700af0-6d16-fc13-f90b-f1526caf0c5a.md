
# ColorFormat.Type Property (Word)

Returns or sets the shape color type. Read-only  **MsoColorType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[ColorFormat](5f12793f-d847-ecf2-6cf6-39387f7f0b28.md)** object.
