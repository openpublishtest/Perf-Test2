
# DataLabel.Shadow Property (Word)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[DataLabel](b955596d-ac94-1e18-4e72-cdf090fc1f9e.md)** object.
