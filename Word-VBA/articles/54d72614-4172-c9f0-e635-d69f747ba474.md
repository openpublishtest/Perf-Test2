
# LinkFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **LinkFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[LinkFormat](ca37d4e2-e978-8e6a-1e7a-7e43cf41e6c2.md)** object.
