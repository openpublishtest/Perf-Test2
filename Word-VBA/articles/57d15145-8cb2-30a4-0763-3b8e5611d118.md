
# OMathBreaks.Count Property (Word)

Returns the number of items in the  **OMathBreaks** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[OMathBreaks](fa01cd62-b8ad-52bf-f36a-f5d1548d3d1e.md)** object.
