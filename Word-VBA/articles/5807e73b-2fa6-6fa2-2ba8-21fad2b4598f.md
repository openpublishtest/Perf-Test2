
# MailMessage.GoToPrevious Method (Word)

Displays the previous mail message if you are using Word as your e-mail editor.


## Syntax

 _expression_ . **GoToPrevious**

 _expression_ Required. A variable that represents a **[MailMessage](d0109969-27f7-0180-c56d-5b49a3f0171b.md)** object.
