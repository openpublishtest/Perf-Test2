
# Range.ID Property (Word)

Returns or sets the identification name for the specified range. Read/write  **String** .


## Syntax

 _expression_ . **ID**

 _expression_ Required. A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.
