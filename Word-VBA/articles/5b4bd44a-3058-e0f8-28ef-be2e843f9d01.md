
# StyleSheet.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[StyleSheet](5e576ff8-c458-f5bd-730d-9db827c4f76e.md)** object.
