
# UpBars.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[UpBars](22dff1d2-8f1b-8c48-354c-570906e0f830.md)** object.
