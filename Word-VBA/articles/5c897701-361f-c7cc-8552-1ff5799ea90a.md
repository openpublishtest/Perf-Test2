
# ErrorBars.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents an **[ErrorBars](33949dd1-48fd-9fff-0bec-1439b65d8e04.md)** object.
