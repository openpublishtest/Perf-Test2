
# WebOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **WebOptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[WebOptions](658ae89d-3f92-067b-1309-7fc90b257111.md)** collection.
