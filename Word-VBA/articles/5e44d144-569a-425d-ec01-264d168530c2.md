
# Index.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Index** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[Index](6a2aab98-485b-01c3-8d9b-9e108b455e22.md)** object.
