
# Point.Delete Method (Word)

Deletes the object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **[Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.


## Remarks

Deleting a  **Point** object deletes the entire series.
