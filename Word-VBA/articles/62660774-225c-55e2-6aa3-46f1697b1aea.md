
# FileConverters.Count Property (Word)

Returns a  **Long** that represents the number of file converters in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[FileConverters](b9b8fc53-1c8e-224d-726a-4edf172ca647.md)** collection.
