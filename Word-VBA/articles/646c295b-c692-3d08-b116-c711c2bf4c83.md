
# XMLNamespaces.Count Property (Word)

Returns a  **Long** that represents the number of XML namespaces in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[XMLNamespaces](f11a6cc1-f33d-e1ab-870c-aa5857d66797.md)** collection.
