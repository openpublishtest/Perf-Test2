
# BuildingBlock.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **BuildingBlock** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[BuildingBlock](2558b89f-8552-bb71-fa40-101cab2635ba.md)** object.
