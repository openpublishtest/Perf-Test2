
# PictureFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **PictureFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[PictureFormat](79556e36-81bb-f8df-45ef-c040df709497.md)** object.
