
# Paragraphs.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Paragraphs** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Paragraphs](bdc7a183-2a98-7d47-c86a-5cecd6c91449.md)** collection.
