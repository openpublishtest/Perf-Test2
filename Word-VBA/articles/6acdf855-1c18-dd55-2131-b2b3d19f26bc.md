
# Footnotes.Count Property (Word)

Returns a  **Long** that represents the number of footnotes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Footnotes](d46a0972-2784-4814-d547-30122a35cdc1.md)** collection.
