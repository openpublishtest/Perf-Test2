
# Window.Type Property (Word)

Returns the window type. Read-only  **[WdWindowType](0e1fc1d4-6f17-ceaa-9819-9ed6a4ec83b6.md)** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[Window](d92f83f9-ae44-56c0-4584-7a9359253c6d.md)** object.
