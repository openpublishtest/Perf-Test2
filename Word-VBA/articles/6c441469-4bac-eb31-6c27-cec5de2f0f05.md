
# DataLabel.ReadingOrder Property (Word)

Returns or sets an  **[XlReadingOrder](eb7796d5-8ef5-96a4-dcf4-c86153f9a475.md)** constant that represents the reading order for the specified object. Read/write **Long** .


## Syntax

 _expression_ . **ReadingOrder**

 _expression_ A variable that represents a **[DataLabel](b955596d-ac94-1e18-4e72-cdf090fc1f9e.md)** object.
