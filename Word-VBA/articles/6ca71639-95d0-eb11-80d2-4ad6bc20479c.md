
# ChartArea.Shadow Property (Word)

Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[ChartArea](7b3384df-f331-033d-4dfa-ee2ff26111c6.md)** object.
