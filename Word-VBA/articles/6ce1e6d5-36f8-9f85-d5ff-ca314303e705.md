
# TablesOfAuthoritiesCategories.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TablesOfAuthoritiesCategories** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TablesOfAuthoritiesCategories](344b9c42-01d1-805c-6af6-c8301e24b97e.md)** collection.
