
# PlotArea.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[PlotArea](72d30767-7cfc-3063-0b49-f9fbc129a52c.md)** object.
