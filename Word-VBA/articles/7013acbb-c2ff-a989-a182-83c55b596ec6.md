
# Tasks.Count Property (Word)

Returns a  **Long** that represents the number of tasks in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Tasks](ff521e20-8a25-f9f6-dccf-effea9debeb7.md)** collection.
