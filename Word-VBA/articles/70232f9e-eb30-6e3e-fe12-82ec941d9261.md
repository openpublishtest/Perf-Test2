
# DataLabel.Height Property (Word)

Gets or sets the height, in points, of the object. Read/write  **Double**.


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[DataLabel](b955596d-ac94-1e18-4e72-cdf090fc1f9e.md)** object.


### Return Value

Double
