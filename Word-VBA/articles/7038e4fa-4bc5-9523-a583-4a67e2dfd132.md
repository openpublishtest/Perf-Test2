
# CoAuthLock.Application Property (Word)

Returns an [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md) object that represents the Microsoft Word application. Read-only.


## Syntax

 _expression_ . **Application**

 _expression_ An expression that returns a **[CoAuthLock](3efa12b0-1079-c6df-20c1-a66398161c8e.md)** object.
