
# LineFormat.Style Property (Word)

Returns or sets the line format style. Read/write  **MsoLineStyle** .


## Syntax

 _expression_ . **Style**

 _expression_ Required. A variable that represents a **[LineFormat](28fabccb-d03f-3466-9d07-ea3ebc4cdd11.md)** object.
