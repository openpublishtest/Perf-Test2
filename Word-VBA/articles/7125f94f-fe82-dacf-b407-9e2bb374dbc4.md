
# Shape.Delete Method (Word)

Deletes the specified shape node.


## Syntax

 _expression_ . **Delete**( **_Index_** )

 _expression_ Required. A variable that represents a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The number of the shape node to delete.|
