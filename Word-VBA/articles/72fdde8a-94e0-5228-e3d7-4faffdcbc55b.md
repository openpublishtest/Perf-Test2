
# BuildingBlockTypes.Count Property (Word)

Returns the number of items in the  **BuildingBlockTypes** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[BuildingBlockTypes](fb179437-b736-dd99-3aea-125346aa7a3d.md)** object.
