
# Comment.Date Property (Word)

Returns a  **Date** that represents the date and time that a comment was inserted. Read-only.


## Syntax

 _expression_ . **Date**

 _expression_ A variable that represents a **[Comment](0a2841f3-ca3c-8186-afab-f634ebd97d4c.md)** object.
