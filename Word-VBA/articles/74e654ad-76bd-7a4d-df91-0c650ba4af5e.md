
# AutoCaption.Name Property (Word)

Returns or sets the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents an **[AutoCaption](895b5181-d36f-7f63-572a-c2d37c878e17.md)** object.
