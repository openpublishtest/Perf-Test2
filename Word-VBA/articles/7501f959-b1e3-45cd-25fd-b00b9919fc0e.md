
# Column.Width Property (Word)

 Returns or sets the width of the specified column, in points. Read/write **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Column](49d68571-2a57-6795-34b9-eb09aeb43043.md)** object.
