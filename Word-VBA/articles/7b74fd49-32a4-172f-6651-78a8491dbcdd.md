
# TickLabels.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **[TickLabels](d94e90dc-0b0e-f4af-078e-6f2b97729db5.md)** object.
