
# Points.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Points](d0adc45a-7b31-a25e-d96f-f2a098702501.md)** object.
