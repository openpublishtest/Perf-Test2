
# ChartGroup.BinsUnderflowEnabled Property (Word)

Specifies whether a bin for values below the [BinsUnderflowValue](40143963-c9a9-566e-e8aa-722cad0db0fc.md) is enabled. Read/write **Boolean**.


## Syntax

 _expression_ . **BinsUnderflowEnabled**

 _expression_ A variable that represents a **ChartGroup** object.
