
# HeadingStyle.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HeadingStyle** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HeadingStyle](d57e68ce-4c8b-0063-5077-82462451f336.md)** object.
