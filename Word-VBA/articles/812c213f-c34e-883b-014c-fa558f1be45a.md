
# CustomLabel.Delete Method (Word)

Deletes the specified custom label.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
