
# TextColumns.Width Property (Word)

Returns or sets the width of the Word art text effects, in points. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[TextColumns](00b62c93-db7d-00b9-cc84-9a21e427d0cd.md)** collection.
