
# CustomLabel.Height Property (Word)

Returns or sets the height of a specified custom mailing label, in points. Read/write  **Single** .


## Syntax

 _expression_ . **Height**

 _expression_ An expression that returns a **[CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
