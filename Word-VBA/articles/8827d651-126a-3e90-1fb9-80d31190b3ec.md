
# DisplayUnitLabel.Shadow Property (Word)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[DisplayUnitLabel](9b028f6c-fd66-f767-f3d1-09de0fbdc148.md)** object.
