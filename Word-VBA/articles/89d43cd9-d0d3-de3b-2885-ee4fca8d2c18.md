
# LegendKey.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
