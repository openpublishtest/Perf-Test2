
# Options.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Options** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[Options](873b7b99-3fe1-fd89-9ece-a9355cb827dc.md)** collection.
