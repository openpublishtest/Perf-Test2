
# OLEFormat.Activate Method (Word)

Activates the specified  **OLEFormat** object.


## Syntax

 _expression_ . **Activate**

 _expression_ Required. A variable that represents an **[OLEFormat](d4c7aa65-5d3a-0b79-914b-6f908b506f63.md)** object.
