
# SynonymInfo.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **SynonymInfo** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[SynonymInfo](0af2d733-a038-1f67-ddca-2b05b3af1b7c.md)** object.
