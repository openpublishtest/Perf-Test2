
# Range.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Range** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.
