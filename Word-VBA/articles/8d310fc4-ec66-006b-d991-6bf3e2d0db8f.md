
# HTMLDivisions.Count Property (Word)

Returns a  **Long** that represents the number of HTML DIV elements in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents an **[HTMLDivisions](fe896440-817f-5485-794c-c5e9700cd062.md)** collection.
