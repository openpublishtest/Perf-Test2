
# Template.ListTemplates Property (Word)

Returns a  **[ListTemplates](5b5f3ed8-4522-f52e-5ae8-9df26a7da154.md)** collection that represents all the list formats for the specified document, template, or list gallery. Read-only.


## Syntax

 _expression_ . **ListTemplates**

 _expression_ A variable that represents a **[Template](47d1d92d-bba9-3f2a-9c71-22ac43159bd3.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](http://msdn.microsoft.com/library/28f76384-f495-9640-a7c8-10ada3fac727%28Office.15%29.aspx).
