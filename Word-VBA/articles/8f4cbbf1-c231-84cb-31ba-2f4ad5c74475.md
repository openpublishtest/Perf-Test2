
# Walls.Name Property (Word)

Returns the name of the object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **[Walls](e98c7218-b944-12bb-caf9-daecee4b6c0c.md)** object.
