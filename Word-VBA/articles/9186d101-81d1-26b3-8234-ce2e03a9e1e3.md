
# InlineShape.Borders Property (Word)

Returns a  **[Borders](6dd1d4cc-2dcf-22c7-a299-4721a5543ba3.md)** collection that represents all the borders for the specified shape.


## Syntax

 _expression_ . **Borders**

 _expression_ Required. A variable that represents an **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](http://msdn.microsoft.com/library/28f76384-f495-9640-a7c8-10ada3fac727%28Office.15%29.aspx).
