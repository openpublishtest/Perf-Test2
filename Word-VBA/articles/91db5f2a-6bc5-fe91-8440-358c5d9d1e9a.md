
# Sections.Count Property (Word)

Returns a  **Long** that represents the number of sections in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
