
# Words.First Property (Word)

Returns a  **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object that represents the first word in a collection of words.


## Syntax

 _expression_ . **First**

 _expression_ Required. A variable that represents a **[Words](a718f69f-1db1-231a-9d65-bf20b48778ed.md)** collection.
