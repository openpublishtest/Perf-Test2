
# ReadabilityStatistics.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ReadabilityStatistics** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ReadabilityStatistics](eabef73c-f837-435a-cfec-b76082cc0f7e.md)** collection.
