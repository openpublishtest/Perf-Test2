
# OMathMatCol.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathMatCol** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathMatCol](5ba2c52f-d190-0b0e-41e7-3e20a52ac77e.md)** object.
