
# DropDown.Value Property (Word)

Returns or sets the number of the selected item in a drop-down form field. Read/write  **Long** .


## Syntax

 _expression_ . **Value**

 _expression_ Required. A variable that represents a **[DropDown](55233d61-d6d0-30f9-6825-ebbdbeb928b6.md)** object.
