
# Subdocuments.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Subdocuments** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Subdocuments](8e14fced-19b3-2794-39a3-9e5ec15dd0ad.md)** collection.
