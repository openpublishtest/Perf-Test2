
# MailMergeFieldNames.Count Property (Word)

Returns a  **Long** that represents the number of mail merge field names in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[MailMergeFieldNames](5a3752da-63b2-f0f9-7456-01a31bac5f62.md)** collection.
