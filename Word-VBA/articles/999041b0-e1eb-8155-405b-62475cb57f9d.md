
# Find.Format Property (Word)

 **True** if formatting is included in the find operation. Read/write **Boolean** .


## Syntax

 _expression_ . **Format**

 _expression_ Required. A variable that represents a **[Find](da822788-cad5-992a-a835-18cc574cc324.md)** object.


## Example

This example removes all bold formatting in the active document.


```vb
With ActiveDocument.Content.Find 
 .ClearFormatting 
 .Font.Bold = True 
 .Format = True 
 .Replacement.ClearFormatting 
 .Replacement.Font.Bold = False 
 .Execute Forward:=True, Replace:=wdReplaceAll, _ 
 FindText:="", ReplaceWith:="" 
End With
```
