
# Gridlines.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[GridLines](9dc77c2a-854f-63c0-4648-b7802fb6d9a2.md)** object.
