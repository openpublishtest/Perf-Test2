
# Rectangle.Range Property (Word)

Returns a  **Range** object that represents the portion of a document that is contained within a rectangle.


## Syntax

 _expression_ . **Range**

 _expression_ Required. A variable that represents a **[Rectangle](90ad4f48-2051-38f9-9b2e-a14bd38478be.md)** object.
