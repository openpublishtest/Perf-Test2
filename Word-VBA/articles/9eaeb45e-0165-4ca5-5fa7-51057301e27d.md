
# HTMLDivisions.Item Method (Word)

Returns an individual  **HTMLDivision** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents an **[HTMLDivisions](fe896440-817f-5485-794c-c5e9700cd062.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

HTMLDivision
