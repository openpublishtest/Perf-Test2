
# XMLNamespaces.Item Method (Word)

Returns an individual  **XMLNamespace** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[XMLNamespaces](f11a6cc1-f33d-e1ab-870c-aa5857d66797.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

XMLNamespace
