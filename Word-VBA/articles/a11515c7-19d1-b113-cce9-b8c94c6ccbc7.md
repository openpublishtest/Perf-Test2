
# ShapeRange.Height Property (Word)

Returns or sets the height of the specified shape range. Read/write  **Single** .


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
