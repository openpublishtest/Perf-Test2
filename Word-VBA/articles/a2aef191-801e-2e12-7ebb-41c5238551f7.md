
# Comments.Count Property (Word)

Returns a  **Long** that represents the number of items in the **Comments** collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Comments](e384b37a-50e3-a214-52a8-6fda2acc4991.md)** collection.
