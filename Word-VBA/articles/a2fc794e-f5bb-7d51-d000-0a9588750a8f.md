
# Windows.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Windows** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Windows](377b493b-e73c-0132-869c-3876c3beaef7.md)** collection.
