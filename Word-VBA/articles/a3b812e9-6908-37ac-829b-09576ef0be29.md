
# ShadowFormat.Blur Property (Word)

Returns or sets a  **Single** that represents the blur level for a shadow format. Read/write.


## Syntax

 _expression_ . **Blur**

 _expression_ An expression that returns a **[ShadowFormat](2a179f0b-ec18-c3dd-dd73-51b18f42e0e2.md)** object.
