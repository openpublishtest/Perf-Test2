
# ChartGroup.BinsType Property (Word)

Specifies how the horizontal axis of the histogram chart is formatted, by bins type. Read/write [XlBinsType](945e729b-f0a0-fc0f-d198-c85aab081d7e.md).


## Syntax

 _expression_ . **BinsType**

 _expression_ A variable that represents a **ChartGroup** object.
