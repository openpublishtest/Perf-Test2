
# ChartFont.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartFont](2ca7fb97-fa22-dec1-6978-8ebb6d8aad7c.md)** object.
