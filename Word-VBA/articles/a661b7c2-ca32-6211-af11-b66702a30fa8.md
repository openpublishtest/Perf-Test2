
# DataLabels.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **[DataLabels](a7676f18-b1f2-1e11-9489-863cb85c1669.md)** object.
