
# Axes.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[Axes](57261ca9-7fd6-ba99-19bd-5df8e940f714.md)** object.
