
# FormField.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FormField** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[FormField](c3c07344-06b2-fe86-6fcb-b9c63a991bcc.md)** object.
