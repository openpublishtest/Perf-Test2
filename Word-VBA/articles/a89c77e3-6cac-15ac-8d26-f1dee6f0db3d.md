
# ReflectionFormat.Offset Property (Word)

Returns or sets the amount of separation, in points, of the reflected image from the shape. Read/write.


## Syntax

 _expression_ . **Offset**

 _expression_ An expression that returns a **[ReflectionFormat](ee2ea7a1-0671-bbd6-6a94-90f2472d704e.md)** object.


### Return Value

Single
