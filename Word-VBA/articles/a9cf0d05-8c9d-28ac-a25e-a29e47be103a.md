
# XMLNamespace.Delete Method (Word)

Deletes the specified XML schema from the list of available XML schemas.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[XMLNamespace](a50ed533-8961-f433-d501-797e5d678fda.md)** object.
