
# Section.Range Property (Word)

Returns a  **Range** object that represents the portion of a document that's contained in the specified object.


## Syntax

 _expression_ . **Range**

 _expression_ Required. A variable that represents a **[Section](3fe563d8-fc05-c17a-e67b-c50eea7e7f13.md)** object.


## Example

This example inserts text at the end of section one.


```vb
Set myRange = ActiveDocument.Sections(1).Range 
With myRange 
 .MoveEnd Unit:=wdCharacter, Count:=-1 
 .Collapse Direction:=wdCollapseEnd 
 .InsertParagraphAfter 
 .InsertAfter "End of section" 
End With
```
