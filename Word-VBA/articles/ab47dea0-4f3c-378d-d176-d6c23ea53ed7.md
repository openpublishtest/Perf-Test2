
# Fields.Count Property (Word)

Returns a  **Long** that represents the number of fields in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Fields](c79065bb-ba29-22fd-a9d7-90bb10550035.md)** collection.
