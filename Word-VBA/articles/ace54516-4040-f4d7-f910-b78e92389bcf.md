
# Font.Glow Property (Word)

Returns a [GlowFormat](d177e399-cafc-132f-b135-b8b578e76889.md) object that represents the glow formatting for the font used by the specified range of text. Read-only.


## Syntax

 _expression_ . **Glow**

 _expression_ An expression that returns a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
