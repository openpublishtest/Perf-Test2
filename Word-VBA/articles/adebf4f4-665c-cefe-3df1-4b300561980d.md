
# Broadcast.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Broadcast** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **Broadcast** object.


## Property value

 **OBJECT**
