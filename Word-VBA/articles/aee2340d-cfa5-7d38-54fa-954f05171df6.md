
# HeadingStyles.Count Property (Word)

Returns a  **Long** that represents the number of heading styles in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[HeadingStyles](be882a12-1264-8f7e-415b-b8bcbf28e703.md)** collection.
