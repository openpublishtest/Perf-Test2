
# Source.Tag Property (Word)

Returns a  **String** that represents an internal identifying label for a source. Read-only.


## Syntax

 _expression_ . **Tag**

 _expression_ An expression that returns a **[Source](f90108a8-6432-a700-86ce-7b8f9e9c034b.md)** object.
