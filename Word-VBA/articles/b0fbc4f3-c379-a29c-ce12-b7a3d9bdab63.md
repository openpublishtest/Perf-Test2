
# CaptionLabels.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CaptionLabels** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[CaptionLabels](7d18c0d6-6d58-9841-4665-ab13e2e2ad9f.md)** collection.
