
# Index.Range Property (Word)

Returns a  **Range** object that represents the portion of a document that is contained within an index.


## Syntax

 _expression_ . **Range**

 _expression_ Required. A variable that represents an **[Index](6a2aab98-485b-01c3-8d9b-9e108b455e22.md)** object.
