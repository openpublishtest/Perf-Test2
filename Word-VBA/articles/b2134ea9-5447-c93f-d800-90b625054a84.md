
# Point.Format Property (Word)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** .


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **[Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.
