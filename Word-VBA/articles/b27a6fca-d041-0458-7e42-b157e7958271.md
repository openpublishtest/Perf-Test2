
# SeriesLines.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents a **[SeriesLines](7521c592-c5aa-8e50-6268-840a41b3a282.md)** object.
