
# Frameset.Type Property (Word)

Returns the  **Frameset** object type. Read-only **[WdFramesetType](9179515a-ec08-1881-2b85-6623d2836f60.md)** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[Frameset](d76806db-c82f-f7b6-fb85-28a649de48a7.md)** object.
