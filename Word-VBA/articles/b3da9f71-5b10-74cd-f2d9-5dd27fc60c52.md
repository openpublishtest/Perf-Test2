
# Breaks.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Breaks** collection.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[Breaks](c6a9213d-c4bc-a9a6-3889-a32446f1fc25.md)** collection.
