
# Series.AxisGroup Property (Word)

Returns the type of axis group. Read/write  **[XlAxisGroup](ed3ff1ce-28de-165d-bbfa-f3d770f32522.md)** .


## Syntax

 _expression_ . **AxisGroup**

 _expression_ A variable that represents a **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
