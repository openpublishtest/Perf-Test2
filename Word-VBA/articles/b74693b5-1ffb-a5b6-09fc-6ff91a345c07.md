
# Source.Cited Property (Word)

Returns a  **Boolean** that represents whether a source has been cited in a document. Read-only.


## Syntax

 _expression_ . **Cited**

 _expression_ An expression that returns a **[Source](f90108a8-6432-a700-86ce-7b8f9e9c034b.md)** object.
