
# Envelope.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Envelope** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[Envelope](03664453-f7fb-f76a-ea60-37e72b53e17c.md)** object.
