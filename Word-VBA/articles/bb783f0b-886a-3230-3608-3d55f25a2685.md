
# LegendEntry.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[LegendEntry](9f793578-cb9b-faa1-f0a1-ea0f9e90dc6f.md)** object.
