
# LeaderLines.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[LeaderLines](ea8805d1-eec7-eaf6-1046-967e28d6bc56.md)** object.
