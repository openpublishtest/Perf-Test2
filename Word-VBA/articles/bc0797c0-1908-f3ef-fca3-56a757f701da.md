
# Template.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[Template](47d1d92d-bba9-3f2a-9c71-22ac43159bd3.md)** object.
