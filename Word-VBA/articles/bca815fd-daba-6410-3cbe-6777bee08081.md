
# DropLines.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[DropLines](4691b002-8512-7cd3-5a20-561232e18d88.md)** object.
