
# HiLoLines.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[HiLoLines](9f1ed891-7e95-8dd0-745a-ce28555284a9.md)** object.
