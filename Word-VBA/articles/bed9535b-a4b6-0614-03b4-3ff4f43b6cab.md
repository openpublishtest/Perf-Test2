
# InlineShape.GroupItems Property (Word)

Returns a  **[GroupShapes](de29d571-476b-fa8b-619e-f7d0181d9756.md)** collection that represents the shapes that are grouped together for an inline shape. Read-only.


## Syntax

 _expression_ . **GroupItems**

 _expression_ An expression that returns an **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
