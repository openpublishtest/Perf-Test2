
# Range.StoryType Property (Word)

Returns the story type for the specified range, selection, or bookmark. Read-only  **[WdStoryType](046fab29-72c8-8b71-4d3a-903e5cb15326.md)** .


## Syntax

 _expression_ . **StoryType**

 _expression_ Required. A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.
