
# ColorFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ColorFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ColorFormat](5f12793f-d847-ecf2-6cf6-39387f7f0b28.md)** object.
