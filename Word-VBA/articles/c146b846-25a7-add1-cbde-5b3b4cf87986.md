
# LineFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **LineFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[LineFormat](28fabccb-d03f-3466-9d07-ea3ebc4cdd11.md)** object.
