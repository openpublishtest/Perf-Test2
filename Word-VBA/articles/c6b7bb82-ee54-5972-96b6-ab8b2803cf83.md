
# MailMergeDataFields.Count Property (Word)

Returns a  **Long** that represents the number of mail merge data fields in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[MailMergeDataFields](a660288d-1a2c-53ec-20d2-c52353be90c8.md)** collection.
