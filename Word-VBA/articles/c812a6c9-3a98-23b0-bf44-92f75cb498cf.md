
# XMLSchemaReferences.Item Method (Word)

Returns an individual  **XMLSchemaReference** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[XMLSchemaReferences](56bef973-805c-c77a-6d2a-54a39fbd1206.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Variant**|The individual object to be returned. Can be a  **Long** indicating the ordinal position or a **String** representing the name of the individual object.|

### Return Value

XMLSchemaReference
