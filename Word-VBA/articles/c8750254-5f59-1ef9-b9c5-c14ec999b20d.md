
# System.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **System** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[System](db15d780-3bbc-9515-a988-ea798777496f.md)** object.
