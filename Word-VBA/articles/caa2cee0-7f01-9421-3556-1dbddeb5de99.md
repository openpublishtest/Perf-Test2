
# OMathBox.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathBox** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathBox](e744ed0f-99de-f13f-766d-5453fb61ed48.md)** object.
