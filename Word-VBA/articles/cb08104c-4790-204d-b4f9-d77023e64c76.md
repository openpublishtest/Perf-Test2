
# XMLSchemaReferences.Count Property (Word)

Returns a  **Long** that represents the number of XML schemas in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[XMLSchemaReferences](56bef973-805c-c77a-6d2a-54a39fbd1206.md)** collection.
