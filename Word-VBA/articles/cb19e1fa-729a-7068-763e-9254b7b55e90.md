
# CalloutFormat.Type Property (Word)

Returns or sets the callout type. Read/write  **MsoCalloutType** .


## Syntax

 _expression_ . **Type**

 _expression_ Required. A variable that represents a **[CalloutFormat](d54764e6-d761-582b-aa0a-baebd3a7cf6a.md)** object.
