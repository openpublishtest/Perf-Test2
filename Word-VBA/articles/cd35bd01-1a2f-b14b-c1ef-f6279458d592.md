
# TableOfFigures.Delete Method (Word)

Deletes the specified table of figures.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[TableOfFigures](0a93d274-cd2e-3d65-48bc-b8fea84ff268.md)** collection.
