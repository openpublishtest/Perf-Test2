
# HeaderFooter.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HeaderFooter** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HeaderFooter](3f2f926a-9220-5536-80ed-af63d2feb016.md)** object.
