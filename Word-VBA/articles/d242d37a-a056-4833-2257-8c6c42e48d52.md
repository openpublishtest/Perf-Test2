
# TickLabels.Depth Property (Word)

Returns the number of levels of category tick labels. Read-only  **Long** .


## Syntax

 _expression_ . **Depth**

 _expression_ A variable that represents a **[TickLabels](d94e90dc-0b0e-f4af-078e-6f2b97729db5.md)** object.
