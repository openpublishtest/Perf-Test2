
# Words.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Words** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Words](a718f69f-1db1-231a-9d65-bf20b48778ed.md)** collection.
