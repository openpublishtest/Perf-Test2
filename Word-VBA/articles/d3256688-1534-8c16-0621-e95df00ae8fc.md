
# XMLNode.HasChildNodes Property (Word)

Returns a  **Boolean** that represents whether an XML node has child nodes. Read-only.


## Syntax

 _expression_ . **HasChildNodes**

 _expression_ An expression that returns an **XMLNode** object.
