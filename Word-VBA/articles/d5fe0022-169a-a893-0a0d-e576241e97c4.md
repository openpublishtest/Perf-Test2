
# CoAuthUpdate.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CoAuthUpdate** object.


## Syntax

 _expression_ . **Parent**

 _expression_ An expression that returns a **CoAuthUpdate** object.
