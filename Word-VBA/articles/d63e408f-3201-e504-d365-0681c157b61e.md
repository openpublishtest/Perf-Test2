
# Window.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Window** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Window](d92f83f9-ae44-56c0-4584-7a9359253c6d.md)** object.
