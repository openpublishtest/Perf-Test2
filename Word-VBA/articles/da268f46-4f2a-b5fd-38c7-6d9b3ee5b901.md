
# Line.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Line** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Line](1fbc9a15-c677-0f79-4311-9e6de6fc1b27.md)** object.
