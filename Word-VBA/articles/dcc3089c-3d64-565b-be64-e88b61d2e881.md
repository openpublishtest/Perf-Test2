
# OMathMat.RowSpacingRule Property (Word)

Returns or sets a  **[WdOMathSpacingRule](25cb7c5d-2d31-9a26-d89d-3fb1e28a3679.md)** constant that represents the spacing rule for rows in a matrix. Read/write.


## Syntax

 _expression_ . **RowSpacingRule**

 _expression_ An expression that returns an **[OMathMat](40478b6e-18fe-b7b7-d0bc-def1349db56a.md)** object.
