
# InlineShape.Delete Method (Word)

Deletes the specified inline shape.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents an **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
