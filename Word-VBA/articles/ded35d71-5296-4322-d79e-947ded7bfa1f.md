
# LegendKey.Delete Method (Word)

Deletes the object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.


## Remarks

Deleting a  **LegendKey** object deletes the entire series.
