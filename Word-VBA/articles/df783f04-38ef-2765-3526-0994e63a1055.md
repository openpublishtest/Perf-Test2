
# XMLNamespaces.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **XMLNamespaces** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[XMLNamespaces](f11a6cc1-f33d-e1ab-870c-aa5857d66797.md)** collection.
