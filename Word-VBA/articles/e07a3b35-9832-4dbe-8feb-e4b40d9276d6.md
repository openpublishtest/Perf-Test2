
# Zoom Properties (Word)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](1946d7d3-ad39-af1f-106b-0a4d8b231ec6.md)|Returns an  **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object that represents the Microsoft Word application.|
