
# Frameset.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Frameset** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Frameset](d76806db-c82f-f7b6-fb85-28a649de48a7.md)** object.
