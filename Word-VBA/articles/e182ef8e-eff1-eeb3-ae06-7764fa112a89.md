
# Axes.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents an **[Axes](57261ca9-7fd6-ba99-19bd-5df8e940f714.md)** object.
