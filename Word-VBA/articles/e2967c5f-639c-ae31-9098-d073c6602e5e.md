
# ChartCategory.Name Property (Word)

Returns or sets a  **String** that represents the name of a chart category. Read/write.


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **ChartCategory** object.


## Property value

 **STRING**
