
# OMathAutoCorrect.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathAutoCorrect** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathAutoCorrect](933b77bd-335e-4f4c-026d-8141bd5fc6e2.md)** object.
