
# Series.ParentDataLabelOption Property (Word)

Specifies the parent data label option (banner, overlapping, or none) for the specified series within the chart group. Read/write [XLParentDataLabelOptions](c83fe64d-5a14-74b5-5847-62cba83805b0.md).


## Syntax

 _expression_ . **ParentDataLabelOption**

 _expression_ A variable that represents a **Series** object.
