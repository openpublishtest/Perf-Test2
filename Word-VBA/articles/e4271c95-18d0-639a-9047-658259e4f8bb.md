
# Shape.Width Property (Word)

Returns or sets the width, in points, of the specified shape. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.
