
# RecentFiles.Count Property (Word)

Returns a  **Long** that represents the number of recent files in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[RecentFiles](c2d5e0b1-0d79-2fa7-c475-e5cace59ba1f.md)** collection.
