
# OMathPhantom.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathPhantom** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathPhantom](8aed370a-98bd-adda-11c1-2df7c213e0f3.md)** object.
