
# Series.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents a **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
