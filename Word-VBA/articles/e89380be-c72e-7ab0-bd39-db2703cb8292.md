
# Border.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Border** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[Border](be48c020-b86c-c004-ce1c-76d9edae9791.md)** object.
