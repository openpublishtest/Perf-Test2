
# Trendline.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Trendline](1cfe897f-26ad-a838-ed9b-f3fd945ff7ea.md)** object.
