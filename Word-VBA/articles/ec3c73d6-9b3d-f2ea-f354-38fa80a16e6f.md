
# ListGallery.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ListGallery** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ListGallery](4fa3af33-becd-0dfc-5c7a-a0e70714e045.md)** object.
