
# ContentControl.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ContentControl** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ContentControl](783dec26-9b63-11f8-6187-985f9c815f27.md)** object.
