
# ChartTitle.Width Property (Word)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.
