
# MailMergeFields.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MailMergeFields** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[MailMergeFields](9d2dfd45-c52b-500e-15bf-1e678e6c1e92.md)** collection.
