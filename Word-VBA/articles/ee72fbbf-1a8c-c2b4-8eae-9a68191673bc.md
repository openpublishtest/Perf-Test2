
# OMathScrSub.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathScrSub** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathScrSub](823053c2-3657-bc4f-6576-7e3560fee188.md)** object.
