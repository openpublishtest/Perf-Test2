
# ReadabilityStatistic.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ReadabilityStatistic** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ReadabilityStatistic](5e82c44d-fc6d-9586-816b-0c46c4a01f3b.md)** object.
