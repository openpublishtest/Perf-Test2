
# ShapeRange.Count Property (Word)

Returns a  **Long** that represents the number of shapes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
