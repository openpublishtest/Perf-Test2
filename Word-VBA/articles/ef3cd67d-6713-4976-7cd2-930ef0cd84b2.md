
# FormFields.Count Property (Word)

Returns a  **Long** that represents the number of fields in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[FormFields](a44a0f57-123b-cade-e306-ba6dc179b619.md)** collection.
