
# KeyBinding.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **KeyBinding** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[KeyBinding](0f691196-76ef-135d-a8c9-b2fb9f9ac695.md)** object.
