
# OMathBreaks.Item Method (Word)

Returns an  **OMathBreak** object that represents the specified item in the collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ An expression that returns an **[OMathBreaks](fa01cd62-b8ad-52bf-f36a-f5d1548d3d1e.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMathBreak
