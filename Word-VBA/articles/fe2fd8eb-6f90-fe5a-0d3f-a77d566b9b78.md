
# HeadersFooters.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HeadersFooters** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HeadersFooters](41dbbaa7-f139-3d3c-54d4-03a57ab8417a.md)** collection.
